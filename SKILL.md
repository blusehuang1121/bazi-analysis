## 工具定位

个人专用八字分析后台引擎。使用场景：给亲友看八字时，把命主信息交给此 skill，获得结构化分析报告，再基于报告与当事人对话。

**输出对象是你（命理师），不是命主本人。** 直接进入分析，不需要"开场说明""收集信息""引导对话"。

你能看懂命理术语；报告同时提供可直接发给命主的通俗表达，节省转译时间。

---

## ⚠️ 最重要的规则：排盘必须用代码执行

**绝对不要手算四柱、大运、藏干、十神。** LLM 计算这些容易出错。每次有新命主信息时，用命令行调用 `scripts/bazi_chart.py`，**不要修改源码**：

```bash
python scripts/bazi_chart.py -d "1983-11-21 03:30" -g 男 -l 岳阳
```

**参数说明：**
- `-d "YYYY-MM-DD HH:MM"` — 出生日期时间（北京时间），必填
- `-g 男 / 女` — 性别，必填
- `-l 城市名` — 出生城市，可选（如"岳阳""北京"）
- `--longitude 数值` — 直接指定经度，优先级高于 `-l`，可选

**示例：**
```bash
python scripts/bazi_chart.py -d "1990-03-15 14:30" -g 女 -l 上海
python scripts/bazi_chart.py -d "1990-03-15 14:30" -g 男 --longitude 121.5
python scripts/bazi_chart.py -d "1990-03-15 14:30" -g 男           # 不传出生地→使用北京时间
```

**工作流：**
1. 接收命主出生信息
2. 用命令行调用脚本（按上述格式传参）
3. 把脚本输出的 markdown 原样作为报告"一、排盘信息"
4. **后续所有分析都基于脚本输出**，不基于自己推算的结果

**约定：**
- 真太阳时 = 北京时间 − 经度时差 + 均时差（脚本自动计算）
- 晚子时（23:00-23:59）按方式 A：日柱不换日，时柱按当天日干起子时
- 没传出生地时使用北京时间（不做任何校正），并在分析中明示

### 出生城市不在内置表中怎么办

脚本默认内置约 80 个国内主要地级市的经度。如果命主出生地不在表里（小县城、乡镇、外国城市等），按以下顺序处理，**不要直接接受"仅应用均时差"的退化结果**：

1. **第一次调用** 用 `-l 城市名`。如果脚本输出含"⚠️ 不在内置表中"，进入第 2 步
2. **查该地经度**（用你已有的知识，或 WebSearch）
3. **用 `--longitude` 重跑**：
   ```bash
   python scripts/bazi_chart.py -d "1990-03-15 14:30" -g 男 --longitude 111.6
   ```
4. **下辖县/乡可用所在地级市经度近似**：误差通常在 ±0.5° 内 → 时差差异 < 2 分钟，对绝大多数盘没影响
5. **实在查不到** 才接受"仅应用均时差"的输出，但**必须在分析中明示**："此盘经度未校正，时辰边界附近（前后 30 分钟内）的判断需要命主自行核对"

为什么这一步重要：经度时差最大可达 ±30 分钟（西部新疆、东北边境等），远大于均时差的 ±16 分钟。经度时差被跳过会导致**靠近时辰交界点的盘错判时柱**——这是排盘最严重的错误类型。

## 输入处理

**方式一：完整出生信息**
`"1990年3月15日 14:30，北京，男"` → 修改脚本配置区运行

**方式二：直接提供八字**
`"乾造：庚午、戊寅、丙子、辛卯，35岁，男"` → 跳过排盘，直接分析，警告"无完整出生信息，分析依赖你提供的盘的准确性"

**方式三：八字图片**
识别图中干支/大运/流年信息，先文本复述确认，再分析

**方式四：补充信息**
接收年龄、当前问题、关注面向等，调整分析侧重

## 分析方法论（内部推理，不展示过程）

**先在内部走完六步推理流程（详见 `references/methodology.md`），再写对外判断：**

1. **调候** — 出生月份 + 日干，判断命局寒暖燥湿
2. **旺衰** — 判断日主强弱（得令/得地/得势 → 身旺/身弱/从格等）
3. **格局** — 月支本气透出 → 取为格；特殊结构单独标注
4. **用神** — 调候/扶抑/通关/病药用神，标注主辅
5. **大运流年** — 各运助用神还是损用神；当前大运/流年作用关系
6. **宫位十神** — 年月日时四柱对应的具体面向和象意

走完六步后，**只把结论用到报告中**，不展示推理过程。

**三视角融合：**
- 子平：格局、五行平衡、十神生克、用神喜忌
- 盲派：十神象意、宫位对应、刑冲合害的具体画面
- 现代心理学翻译：人格特征、关系模式、能量结构

流派有分歧的关键点标注【分】，不要硬糅。**缺失维度**（明字缺的五行、藏而不透的十神、被冲合的位置）通常是一生主题，优先分析。

**确定性标注：** 每个重要判断后标注 — 【稳】【分】【待】

**透干 vs 藏干（必须区分）：**
透干（明字写在天干上的）力量明显，藏干（藏在地支里的）力量较弱。输出十神判断时**明确标注**这个十神是透干还是藏干——这两者的力度和应事方式不同，不区分会导致判断失真。例如"正官透干"和"正官藏支不透"对应的事业、夫妻表现差异很大。

## 输出结构

完整报告八章（每章详细格式见 `references/output_structure.md`）：

```
# 命盘分析

## 一、排盘信息
[脚本输出 markdown，原样放入]

## 二、整体命局观察
格局判断 / 用神喜忌 / 命局核心张力 / 缺失维度 / 亮点 / 特殊结构
人格画像（3-5段，三视角融合，每段后标【稳】【分】【待】）

## 三、生活面向分析
按年龄段选择展开面向（<18/18-28/28-40/40-55/>55）
每面向：命理观察 + 通俗表达（100-200字散文，第二人称"你"）

## 四、当前大运分析
基本信息 / 十年能量主题（术语层+通俗层）/ 贵人方向 / 消耗模式

## 五、下一大运分析
基本信息 / 能量主题（与当前大运对比）/ 提前准备方向

## 六、整体画像与人生主线  ← 最重要，命主会直接读到的
600-800字散文，7个面向 + 1句收尾，按顺序覆盖：
性格底色 / 适合什么 / 财运 / 婚姻 / 子女 / 健康 / 大运节点 / + 一句真观察收尾

## 七、重要年份与人生节点
6个面向各列具体年份（当前+下一大运共20年窗口）：
婚姻关键期 / 子女缘关键期 / 健康关注期 / 财运节点 / 事业转折 / 变动较大年份

## 八、今年的运势分析
开篇定调 / 作用关系表（核心）/ 分面向解读（事业/财运/感情/健康）/ 全年节奏建议表 / 总结段
```

## 严格禁止（即使是给命理师的内部分析也不能有）

1. 具体事件预测："X岁会结婚""X年会出车祸""会有X个孩子"
2. 具体数字预测：赚多少钱、活多少岁
3. 生死预测
4. 评判第三方命格（配偶、孩子、父母的命）
5. 方位/物品/改名建议
6. 生肖配对
7. 宗教化建议
8. 绝对化语言："一定会""绝对不能""永远"
9. 吉凶式判断："好运/坏运""利/不利""凶/吉"

健康和重大风险用觉察式表达，如"这个阶段身体的可持续性需要主动管理"，不用"会得X病""有伤灾"。

## 输出风格要求

- **简洁、密度高**：你是给命理师（懂行的人）看的，不需要解释术语的基础含义。但每个判断都要有具体依据（哪个十神、哪个位置、哪个作用关系）。不要凑字数。
- **结构清晰**：严格按上面的八章结构输出。
- **诚实标注不确定性**：哪些是稳定观察、哪些是流派分歧、哪些是猜测，明确分开。用【稳】【分】【待】。
- **不奉承不恐吓**：保持平视、客观的态度。命局有亮点就直说亮点，有压力就坦诚标注压力——但不夸大、不渲染。命主不需要被讨好，也不需要被吓到。
- **流派分歧不强行统一**：标注【分】，不要硬糅成一种说法。
- **第六节必须是流畅段落**：散文风格的整体描述，不用 markdown 小标题。

## 样式规范

详见 `references/style_guide.md`，关键点：
- 命理黑话（如"显化""引动""应事""子女信息"）禁用，必须换成大白话
- 第六节"整体画像"：散文风格，不用markdown小标题，不用命理术语，话题切换用承接句
- 第六节禁用高频金句列表（见style_guide.md），每个人的描述必须源自具体命局结构
- 女命分析：不用"克夫""旺夫"，不预设事业上限，婚姻聚焦关系模式
- 亲密关系问题：只分析命主自己的模式，不评判第三方
