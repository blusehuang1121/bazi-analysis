## 工具定位

个人专用八字分析后台引擎。使用场景：给亲友看八字时，把命主信息交给此 skill，获得结构化分析报告，再基于报告与当事人对话。

**输出对象是你（命理师），不是命主本人。** 直接进入分析，不需要"开场说明""收集信息""引导对话"。

你能看懂命理术语；报告同时提供可直接发给命主的通俗表达，节省转译时间。

### 核心定位：理解整体格局，不预测具体年份

这个工具的目的是帮助命主**理解自己的整体生命格局**，不是预测未来某年某月会发生什么。

具体边界：
- **保留**：脚本算出的流年作用关系完整表（在排盘输出里），用于你（Claude）**内部推理**——它让你的判断有数据支撑，不是凭印象
- **不输出**：对外报告中**不列出具体年份的危机预测**（如"X 年婚姻有危机"），即使脚本算出了高危信号
- **用节奏感代替具体年份**：涉及人生阶段的描述，用"中年这一段会经历一次主动调节""感情上的几个重要节点"这种表达，而不是"2024 年婚姻有危机"
- **唯一例外**：第七节"今年的运势分析"——这是命主当下能感受、能行动的时间窗口，可以具体到今年的能量结构

**为什么这样设计**：

精确预测年份对命主的实际价值很低——知道"2024 婚姻有危机"，命主能做什么？这种信息更多是制造焦虑，而不是帮助决策。

而**理解自己的整体格局**有持久的实际价值：知道"我这个人在感情上一直需要主动经营""我适合做需要持续积累的工作"——这种认知会改变命主很多日常决定，而且不会因为某年算错就失效。

## ⚠️ 最重要的规则：排盘必须用代码执行

**绝对不要手算四柱、大运、藏干、十神、流年作用关系。** LLM 计算这些容易出错。每次有新命主信息时，用命令行调用 `scripts/bazi_chart.py`，**不要修改源码**：

```bash
python scripts/bazi_chart.py -d "1983-11-21 03:30" -g 男 -l 岳阳
```

**参数说明：**
- `-d "YYYY-MM-DD HH:MM"` — 出生日期时间（北京时间），必填
- `-g 男 / 女` — 性别，必填
- `-l 城市名` — 出生城市，可选（如"岳阳""北京"）
- `--longitude 数值` — 直接指定经度，优先级高于 `-l`，可选

**工作流：**
1. 接收命主出生信息
2. 用命令行调用脚本（按上述格式传参）
3. 把脚本输出的 markdown 原样作为报告"一、排盘信息"
4. **后续所有分析（包括流年作用关系判断）都基于脚本输出**，不基于自己推算的结果

**脚本输出包含：**
- 基本信息（含真太阳时校正过程；时辰边界变化时输出警告）
- 四柱八字 + 藏干 + 十神
- 五行分布
- 大运排布
- **各大运 vs 原局的作用关系表**
- **流年作用关系完整表（从起运到 80 岁），含 S/A/B/C/D 五档关键信号识别**

最后两张表是 Step 5（大运流年作用关系）和第七节"今年的运势分析"的数据来源——必须读表，不要手算。

**约定：**
- 真太阳时 = 北京时间 − 经度时差 + 均时差（脚本自动计算）
- 晚子时（23:00-23:59）按方式 A：日柱不换日，时柱按当天日干起子时
- 没传出生地时使用北京时间（不做任何校正），并在分析中明示

### 出生城市不在内置表中怎么办

脚本默认内置约 80 个国内主要地级市的经度。如果命主出生地不在表里：

1. 第一次用 `-l 城市名`，如果脚本输出含"⚠️ 不在内置表中"，进入第 2 步
2. 查该地经度（用你已有的知识，或 WebSearch）
3. 用 `--longitude` 重跑：`python scripts/bazi_chart.py -d "..." -g 男 --longitude 111.6`
4. 下辖县/乡可用所在地级市经度近似（误差通常 ±0.5° 内 → 时差 < 2 分钟）
5. 实在查不到才接受"仅应用均时差"的输出，但**必须在分析中明示**"此盘经度未校正，时辰边界附近的判断需要命主自行核对"

**为什么重要**：经度时差最大可达 ±30 分钟，远大于均时差的 ±16 分钟。经度时差被跳过会导致**靠近时辰交界点的盘错判时柱**。

### 时辰边界警告（脚本会自动检测）

某些情况下加上均时差会改变时辰判断（出生时间靠近时辰边界时尤其明显）。脚本会在这种情况下输出**警告信息**，明确显示：
- 只校正经度的话是哪个时辰
- 加上均时差后是哪个时辰

如果命主已经习惯按某个时辰排的盘（比如他自己之前找命理师看过），需要核对脚本输出的"完整真太阳时"和命主习惯的版本是否一致。如果不一致，告诉用户："命主习惯的是 X 时盘，请用那个版本重新排"。

**方式一：完整出生信息**
`"1990年3月15日 14:30，北京，男"` → 直接调用脚本

**方式二：直接提供八字**
`"乾造：庚午、戊寅、丙子、辛卯，35岁，男"` → 跳过排盘，直接分析，警告"无完整出生信息，分析依赖你提供的盘的准确性"

**方式三：八字图片**
识别图中干支/大运/流年信息，先文本复述确认，再分析

**方式四：补充信息**
接收年龄、当前问题、关注面向等，调整分析侧重

## 分析方法论（内部推理，不展示过程）

**先在内部走完七步推理流程（详见 `references/methodology.md`），再写对外判断：**

1. **调候** — 出生月份 + 日干，判断命局寒暖燥湿
2. **旺衰** — 判断日主强弱（得令/得地/得势 → 身旺/身弱/从格等）
3. **格局** — 月支本气透出 → 取为格；特殊结构（建禄/羊刃等）单独判定
4. **用神** — 调候/扶抑/通关/病药用神，标注主辅
5. **大运流年作用** — **直接读脚本输出的两张作用关系表**，不要手算
6. **宫位十神** — 年月日时四柱对应的具体面向和象意
7. **盲派做功效率**（财运面向必走）— 真神 / 家内外 / 做功效率 / 三宝 / 引动比劫

走完七步后，**只把结论用到报告中**，不展示推理过程。

**两个关键元规则（详见 methodology.md）：**

- **元规则一：隐性结构优先于显性结构** — 涉及"状态型"判断（哪一年婚姻最不稳、健康最差、心理最容易失控）时，必须扫描天干层面的隐性结构（比劫剋财、伤官见官、七杀被合、用神被合等），不能只看地支冲合
- **元规则二：识别主动型 vs 被动型人格** — 判断"动"的模式时必须先识别命主驱动力类型，**贯穿整体画像的调性**。被动型命主的事业宫和关系宫的动可以跨界表现

**三视角融合：**
- 子平：格局、五行平衡、十神生克、用神喜忌
- 盲派：十神象意、宫位对应、刑冲合害的具体画面、做功效率
- 现代心理学翻译：人格特征、关系模式、能量结构

流派有分歧的关键点标注【分】，不要硬糅。**缺失维度**（明字缺的五行、藏而不透的十神、被冲合的位置）通常是一生主题，优先分析。

**确定性标注：** 每个重要判断后标注 — 【稳】【分】【待】

**透干 vs 藏干（必须区分）：**
透干（明字写在天干上的）力量明显，藏干（藏在地支里的）力量较弱。输出十神判断时**明确标注**这个十神是透干还是藏干——这两者的力度和应事方式不同，不区分会导致判断失真。

## 输出结构

完整报告共 **7 章**（每章详细格式见 `references/output_structure.md`）：

```
# 命盘分析

## 一、排盘信息
[脚本输出 markdown，原样放入]

## 二、整体命局观察
格局判断 / 用神喜忌 / 命局核心张力 / 缺失维度 / 亮点 / 真神判断 / 特殊结构
人格画像（3-5段，三视角融合，每段后标【稳】【分】【待】）

## 三、生活面向分析
按命主年龄段选择展开面向，每面向含命理观察+通俗表达
财运面向必须走 Step 7 盲派做功方法

## 四、当前大运分析
基本信息（读脚本表）/ 十年能量主题（术语层+通俗层）/ 贵人方向 / 消耗模式

## 五、下一大运分析
基本信息 / 能量主题（与当前大运对比）/ 提前准备方向

## 六、整体画像与人生主线  ← 最重要，命主会直接读到的
600-800字散文，7个面向 + 1句收尾
⚠️ 涉及人生时间用节奏感，不用具体年份
⚠️ 调性符合命主人格驱动力类型

## 七、今年的运势分析  ← 唯一可以涉及具体年份的章节
开篇定调 / 作用关系表（核心，读脚本表）/ 分面向解读 / 全年节奏建议表 / 总结段
```

**注意：之前版本里的"七、重要年份与人生节点"已删除**——具体年份判断只在脚本输出的流年作用关系完整表里作为内部参考，对外报告不列出（这是工具的核心定位）。

## 严格禁止（即使是给命理师的内部分析也不能有）

1. **具体事件预测**："X 岁会结婚""X 年会出车祸""会有 X 个孩子"
2. **具体数字预测**：赚多少钱、活多少岁
3. **生死预测**
4. **评判第三方命格**（配偶、孩子、父母的命）
5. **方位/物品/改名建议**
6. **生肖配对**
7. **宗教化建议**
8. **绝对化语言**："一定会""绝对不能""永远"
9. **吉凶式判断**："好运/坏运""利/不利""凶/吉"
10. **盲派传统威胁性判语**：
    - "克夫""克妻""克兄弟""克父母"——不评判任何亲属关系上的"克"
    - "行、吃、急、予、瘫"五种残疾——绝对不与命局结构挂钩
    - "巨富""大富""贫命""乞丐"——不做财富量级的绝对判定
    - "刑伤""灾煞应期"——具体灾难性预测一律禁用
    - 任何把命局结构直接对应到具体灾难/残疾/重大事件的判语
11. **过度自信的财富/事业层级定盘**：不说"处级及以上""年薪 X 百万级别"等具体职级和数字

健康和重大风险用觉察式表达，如"这个阶段身体的可持续性需要主动管理"，不用"会得 X 病""有伤灾"。

## 输出风格要求

- **简洁、密度高**：你是给命理师（懂行的人）看的，不需要解释术语的基础含义。但每个判断都要有具体依据（哪个十神、哪个位置、哪个作用关系）。不要凑字数。
- **结构清晰**：严格按上面的七章结构输出。
- **诚实标注不确定性**：哪些是稳定观察、哪些是流派分歧、哪些是猜测，明确分开。用【稳】【分】【待】。
- **不奉承不恐吓**：保持平视、客观的态度。命局有亮点就直说亮点，有压力就坦诚标注压力——但不夸大、不渲染。命主不需要被讨好，也不需要被吓到。
- **流派分歧不强行统一**：标注【分】，不要硬糅成一种说法。
- **第六节必须是流畅段落**：散文风格的整体描述，不用 markdown 小标题。

## 样式规范

详见 `references/style_guide.md`，关键点：
- 命理黑话（如"显化""引动""应事""子女信息"）禁用，必须换成大白话
- 第六节"整体画像"：散文风格，不用 markdown 小标题，不用命理术语，话题切换用承接句
- 第六节禁用高频金句列表，每个人的描述必须源自具体命局结构
- **第六节涉及人生时间必须用节奏感语言（中年这一段、X 岁前后这几年），不能出现具体公历年份**
- 女命分析：不用"克夫""旺夫"，不预设事业上限，婚姻聚焦关系模式
- 亲密关系问题：只分析命主自己的模式，不评判第三方
- **盲派威胁性判语全部禁用**（克夫/克妻/五种残疾/贫命/巨富/刑伤等）
