# 各章节详细格式

## 二、整体命局观察

### 命局核心特征（术语层，给命理师看）
列点输出：
- 格局判断（取格名称 + 简要说明格的结构）
- 用神喜忌（主用神 + 辅用神，标注是否在原局得力）
- 命局核心张力（核心的生克关系，一两句话）
- 缺失维度（明字缺的五行 / 藏而不透的十神 / 被冲合的关键位置）
- 命局亮点（特别有力的结构）
- 特殊结构（伏吟、反吟、天克地冲、双合等，如有才写）

### 人格画像（三视角融合）
3-5 段，每段聚焦一个核心特征，每段后标注【稳】【分】或【待】。

**融合三视角写：**
- 子平视角的命理判断是"骨"
- 盲派视角的象意和画面是"肉"
- 现代心理学翻译是"翻译层"，让判断更贴近人的实际体感

**注意：** 这一节是给命理师看的判断层，不需要单独写通俗表达。给命主的通俗整合在第六节完成。这样分工清晰、不重复。

## 三、生活面向分析

### 按年龄段选择展开面向

| 年龄段 | 展开面向 |
|---|---|
| 18岁以下 | 学业、性格、家庭关系、潜在天赋 |
| 18-28岁 | 学业收尾、工作起步、性格、感情萌芽、当前大运 |
| 28-40岁 | 工作发展、婚姻、子女（如已婚有子女）、财运、当前+下一大运 |
| 40-55岁 | 工作转型、婚姻状态、子女、财运、健康、大运转换 |
| 55岁以上 | 人生整合、健康、子女、晚年方向、财富守成 |

### 每个面向的格式

```
### [面向名称]
**命理观察**：[术语+判断，给命理师看。明确标注透干还是藏干]
**通俗表达**：[100-200字连贯段落，大白话，第二人称"你"，不用列表]
[确定性标注]【稳】【分】或【待】
```

## 四、当前大运分析

### 大运基本信息
直接从排盘脚本输出取：
- 干支、起止年份、已走年数、剩余年数
- 大运天干十神性质
- 与原局的作用关系（合/冲/生/克，具体干支）

### 这十年的能量主题

**给命理师的版本**：用术语描述命理结构和作用关系

**通俗版本**：100-200字连贯段落，大白话描述这十年是什么样的阶段、有什么能量主线、命主可能的体感是什么。不用术语，不用列表，流畅段落。

### 这个阶段的"贵人"方向
不用宿命式判断，用关系式建议：
- 命局缺什么 + 当前大运需要什么 → 能补这个的人就是贵人
- 描述贵人的"特征类型"而非"生肖/方位"

### 这个阶段需要警惕的消耗模式
觉察式建议，不是禁令：
- 这个大运容易透支什么
- 命局原本的弱点在这个运里是否被放大
- 容易陷入的关系/财务/健康模式

## 五、下一大运分析

### 下一大运基本信息
- 干支、开始年份、天干十神性质（从排盘取）

### 能量主题
与当前大运对比（延续？反向？过渡？）

### 提前准备方向
- 需要提前培养的能力
- 需要开始关注的机会类型
- 需要调整的习惯或模式
- 需要警惕的转折期问题

## 六、整体画像与人生主线（最重要）

这是整份报告里**唯一会被命主完整读到的部分**。它不是前面几节的文学化总结，是**老派命理师那种"一段话讲清这个人一生"的整体描述**——结构清楚、面向齐全、每个面向用一两句话说透，整体精炼有力。

### 写作的核心模型

想象一位经验丰富的老命理师坐在你对面，看完你的盘后用 600-800 字告诉你这一生的全貌。他的语言不文学化、不堆比喻、不反复铺陈。他**按面向逐一说**：你是个什么样的人，你适合做什么事、不适合做什么事，你的财运怎么走，你的婚姻怎么样，你的子女怎么样，你的健康要注意什么，你这辈子的关键节点在哪里。

每个面向**只说一两句到位的话就过**，不在一个面向上反复打转。

### 必须按顺序覆盖的 7 个面向 + 1 句收尾

1. **性格底色**（约80-150字）：你是个什么样的人。这个人最核心的一两个特征
2. **适合什么/不适合什么**（约80-150字）：发力方向、避开方向
3. **财运**（约60-120字）：钱怎么来、什么样的财、需要警惕什么
4. **婚姻/亲密关系**（约60-120字）：你在关系里的模式、配偶大概什么样、关系的张力点
5. **子女/与下一代的关系**（约40-80字，篇幅可以小）：缘分如何、关系模式如何
6. **健康**（约40-80字）：要留意的身体面向
7. **大运节点/人生主线**（约100-150字）：当前大运是什么阶段、下一大运的转换、一生的大致脉络

**收尾**（一两句话）：一句真观察，不是鸡汤

**总长度控制在 600-800 字**。如果某个面向命局信息很弱，那一段就少写或一笔带过——不要为了凑字数硬编。

### 写作技法（7 条）

#### 第一：每个面向只用一两句话说透，不反复铺陈

老派命理师的力量在"一句话定调"。不要在一个面向里写三段话来反复表达同一件事。**说到了，就过。**

#### 第二：话题转换要清晰，不要把所有面向揉成一锅粥

读者要能明显看出"现在开始讲财运了""现在转到婚姻了"。用承接句切换：

- "再说工作和事业。"
- "财运这一块。"
- "婚姻和感情上。"
- "子女方面。"
- "身体上要留意的是。"
- "再讲讲你这辈子的大运节点。"

不要用 markdown 小标题（不能用 `###` 或 `**财运**` 这种），就用承接句完成话题切换。

#### 第三：避免文学化堆砌

不要用比喻堆砌（"像河流一样""像山一样""像火焰一样"），不要用排比铺陈（"你 X，你 Y，你 Z"），不要用形容词堆叠（"深邃的、敏锐的、富有创造力的"）。

**老命理师不写散文**。他说的是结论，不是抒情。

#### 第四：避免命理术语和命理黑话

不能用：伤官、七杀、比劫、用神、化解、显化、引动、应事、信息、结成、透出。这些命主听不懂。

可以用：大运、流年、八字、命局、五行（这些入门级词汇命主基本都能理解）。

#### 第五：敢说硬的部分，但用建设性的方式

如果某个面向有真实的张力（比如婚姻有冲、健康有压力），**直说**，但说完要给一个怎么应对的方向。

例子：
- 不写："你婚姻不好。"
- 也不写："你的婚姻完全没问题。"
- 写："婚姻上有个特点——你的伴侣大概是个有自己主张、不太顺着你的人。这段关系里有张力，但这种张力本身不是坏事，关键是你怎么用它。"

#### 第六：结尾用一句真观察收，不要鸡汤

禁用结尾："愿你..."、"祝你..."、"未来可期"、"加油"。

可以这样结尾：
- "这是你这辈子的一个核心议题，不是一个阶段的事。"
- "走好这十年，下一个阶段你会更舒展。"
- "你现在做的事情，是为五年后铺路。"

#### 第七：避免金句复用和同质化表达（最重要）

详细的禁用金句列表见 `style_guide.md`。

**核心原则：** 每次写画像时，**强制基于这个具体盘的具体结构**来构造表达。

不是"这个人内向 → 套用'想得多说得少'"，而是先问："这个人的内向具体来自哪里？"
- 印星深透 → 思考型内向
- 日支孤辰 → 社交型疏离
- 食伤被克 → 表达受阻
- 比劫弱 → 不愿群体取暖

不同来源对应的内向是不一样的——**用不一样的话来描述**。

**自检：** 写完画像后回看，问"这段话里有没有任何一句，是我能直接搬到另一个相似命主身上也能成立的？"如果有，那一句就需要重写——让它**只对这个特定盘成立**。

### 范文骨架示例

⚠️ **下面这个示例只是用来说明结构和节奏**——7 个面向怎么排、承接句怎么用、长度怎么控制。**不要照搬里面的具体句子**，每次都要根据真实命局重新构造表达。

```
[第一段：性格底色，80-150字]
[切入语 + 一两句核心特征 + 一句这个特征的优势/陷阱]

[第二段：适合什么 / 不适合什么，80-150字]
"适合做的事..." 或 "事业上..." 切入
[发力方向 + 避开方向 + 一句对你独特工作模式的描述]

[第三段：财运，60-120字]
"财运这一块..." 或 "钱财上..." 切入
[财气从哪里来 + 求财方式 + 需要警惕什么]

[第四段：婚姻，60-120字]
"婚姻上..." 或 "感情这一块..." 切入
[配偶的大概特征 + 关系里的张力点 + 一句建设性的提醒]

[第五段：子女，40-80字]
"子女方面..." 切入
[子女缘分的形态 + 与下一代关系模式]

[第六段：健康，40-80字]
"身体上..." 切入
[最需要留意的器官/系统 + 容易透支的方式]

[第七段：大运节点和人生主线，100-150字]
"再说说你这辈子的节点..." 切入
[过去 → 当前 → 未来的脉络 + 关键转换点]

[结尾一句]
[真观察，不是鸡汤]
```

**这个骨架是死的，每次填进去的内容必须是活的——基于这个具体盘的具体结构构造。**

## 七、重要年份与人生节点

这一节是命主最关心、市面命理师最常给的实用判断。但**这一类内容最容易滑向"算命预言"**——所以输出格式和措辞必须严格遵守下面的规则。

### 输出原则

**不预测具体事件**，只标注**值得关注的时间窗口和需要留意的方向**。

**错误的写法（禁用）：**
- "你 35 岁会结婚"
- "你 42 岁有车祸"
- "你 50 岁会得重病"
- "你 X 年破财几十万"

**正确的写法：**
- "30-32 岁这几年是感情上的活跃期，关系层面会有比较明显的变化"
- "55 岁前后是身体上需要主动调节的阶段，特别是肝胆方面"
- "45-46 岁前后是事业上的关键转折窗口，这两年的选择影响后面十年"

**核心区别：** 错误的写法说"会发生 X"，正确的写法说"这是 X 类型值得关注的窗口期"。前者是预言，后者是提醒。

### 覆盖范围
**当前大运 + 下一大运**（约 20 年窗口），每个面向列出具体年份。不要只在当前大运里找年份。

### 6 个面向

每个面向都按"判断方法 / 输出要求 / 措辞要点"三层结构来组织。

---

#### 1. 婚姻与亲密关系的关键期

**判断方法：**
- 看哪些流年地支与日支（配偶宫）形成合或冲
- 看哪些流年透出夫妻星（男看财、女看官）
- 看哪些流年遇到桃花、红鸾等情感指标

**输出要求：**
- 列出**当前大运和下一大运里**具体的 2-4 个关键年份（具体到某一年）
- 每个年份用一句话说明这是什么样的"关系激活期"
- 已婚的话，重点标注"关系层面会有起伏"的年份
- 未婚的话，重点标注"感情容易显现"的年份

**措辞要点：**
- 说"X 年（X 岁）流年与日支半合，是感情上的活跃期"
- 不说"X 年会结婚""X 年会离婚"

---

#### 2. 子女缘的关键期与时间倾向

**判断方法：**
- 看哪些流年透出子女星（女看食伤、男看官杀），并且与时柱（子女宫）有作用
- 看哪些流年地支与时支形成合（特别是亥卯未三合木局对乙木日主有"得子"信号等）
- 看哪些流年引动原局子女星

**输出要求：**
- 列出**当前大运和下一大运里**具体的 2-3 个子女缘容易显现的年份
- 已婚有子女的话，标注与子女关系上需要留意的年份（比如冲时支的年份）
- 未育的话，标注子女缘分容易显现的具体年份

**措辞要点：**
- 说"X 年（X 岁）流年透食神且合时支，是子女缘分容易显现的窗口"
- 说"X 年与时支相冲，可能是与子女关系上需要多沟通的一年"
- 不说"X 年会生孩子""会生几个"

---

#### 3. 健康关注期

**判断方法：**
- 看哪些流年与原局四柱形成强冲克（特别是冲日柱、冲日支）
- 看哪些流年加重原局已经偏弱五行的负担（比如水旺命局遇到水更旺的流年）
- 看哪些流年与大运形成天克地冲

**输出要求：**
- 先标注**长期需要关注的器官系统**（基于命局五行受克的固定方向，这个不变）
- 然后列出**当前大运和下一大运里**具体的 2-3 个身体最容易透支的年份
- 每个年份用一句话说明那一年身体上的具体注意点

**措辞要点：**
- 说"X 年（X 岁）流年冲日柱，是身体容易透支、特别是 X 系统需要主动调节的一年"
- 说"X 大运里 X 年和 X 年这两年身体最值得留意"
- 不说"X 年会得 X 病""X 年有伤灾"

---

#### 4. 财运的关键节点

**判断方法：**
- 看哪些流年透出财星（日主所克的五行）并且财星不被冲克
- 看哪些流年比劫旺（容易耗财、合作中的财务波动）
- 看哪些流年财库（辰戌丑未中藏财的位置）被开或被冲

**输出要求：**
- 列出**当前大运和下一大运里**具体的 2-4 个财气活跃的年份
- 同时标注 1-2 个需要警惕财务风险的年份
- 每个年份用一句话说明那一年财气的形态

**措辞要点：**
- 说"X 年（X 岁）流年财星到位，是财气活跃、做事容易转化为收益的一年"
- 说"X 年比劫透干，需要留意合作中的财务分配"
- 不说"X 年会发财多少""X 年会破财多少"

---

#### 5. 事业的关键转折窗口

**判断方法：**
- **大运交脱前后 1-2 年**（这是固定的转折期，必须标注）
- 流年与大运形成天克地冲的年份（重大方向调整期）
- 流年透出官杀且与原局有作用的年份（事业上承担更多责任的年份）
- 流年冲月柱（事业宫）的年份

**输出要求：**
- 列出**当前大运和下一大运里**具体的 3-4 个事业转折点
- 大运交脱年份必须明确标注（比如"2027 年大运交脱，2026-2028 这三年是过渡期"）
- 每个年份用一句话说明那一年事业上的特点

**措辞要点：**
- 说"X 年（X 岁）大运交脱前的过渡期，事业方向上需要做关键判断"
- 说"X 年流年冲月柱，事业方向上容易出现明显变化"
- 不说"X 年会升职""X 年会被裁"

---

#### 6. 变动较大的年份（最敏感，最克制）

**判断方法：**
- 流年与日柱形成天克地冲（"岁运并临"或"日犯太岁"等组合）
- 流年与当前大运形成天克地冲
- 多重作用关系叠加的年份

**输出要求：**
- 列出**当前大运和下一大运里** 1-3 个变动最大的具体年份
- 每个年份说明命理结构特点 + 建议的心态准备
- **绝对不说：** 车祸、意外、伤灾、官司、灾难、重病——这些词都禁用

**措辞要点：**
- 说"X 年（X 岁）流年与日柱形成强烈作用关系，是过山车型的一年，事先做好心理准备和应对方案"
- 说"X 年是变动比较大的一年，建议在这一年不做重大投资和重大方向调整"
- 不说"这一年要小心 X 灾"

---

### 输出格式

```
**婚姻 / 亲密关系的关键年份**
- XXXX 年（X 岁）：[流年作用关系，一句话][这一年关系层面的特点]
- XXXX 年（X 岁）：[同上]

**子女缘的关键年份**
- 长期倾向：[一两句话]
- XXXX 年（X 岁）：[一句话]
- XXXX 年（X 岁）：[一句话]

**健康关注的关键年份**
- 长期需要关注的器官系统：[一两句话，不变的部分]
- XXXX 年（X 岁）：[这一年身体上需要留意什么]
- XXXX 年（X 岁）：[同上]

**财运的关键年份**
- 财气活跃年：XXXX 年（X 岁）：[一句话]
- 财气活跃年：XXXX 年（X 岁）：[一句话]
- 需要留意财务的年：XXXX 年（X 岁）：[一句话]

**事业的关键转折年份**
- XXXX 年（X 岁）：[这一年事业上的特点]
- XXXX 年（X 岁）：[同上]
- 大运交脱过渡期：XXXX-XXXX 年：[一句话]

**变动较大的年份（值得提前心理准备）**
- XXXX 年（X 岁）：[命理结构特点 + 建议的心态准备]
```

### 整段后的"给当事人的话"（100-150 字）

最后写一段总结，用大白话告诉命主：
- 这些年份不是"凶吉预测"，是值得提前留意的窗口期
- 知道窗口期的意义不是恐惧，而是**让自己在这些时间点更主动一些**
- 命理给的是"窗口"，事情会怎么发生还是看人怎么应对
- 重要的人生节点不会突然来，命主自己也会有感觉

- [ ] 涵盖范围是"当前大运 + 下一大运"两个十年的窗口？
- [ ] 措辞做到了"提示窗口期 + 建议主动应对"，而不是"预测命运 + 让人恐惧"？

## 八、今年的运势分析

这一节专门分析当前流年（脚本输出的"当前流年"）对命主的影响。命主最关心的就是"今年怎么样"——这一节要讲清楚。

### 整体输出要求（最关键的质感原则）

**用专业但不堆砌术语的方式写**——每出现一个命理术语（如"正财透干""寅午半合""午未六合"），后面立刻配一个生活化的解读（如"财气浮上水面""喜用力量倍增""贵人相助"）。

**这是这一节质感最重要的来源。** 命理结构是骨，生活化解读是肉，两层缺一不可。只有结构没有解读 → 像八股；只有解读没有结构 → 像玄学；两者并列 → 才是专业又可读的命理报告。

### 标准输出结构（按顺序）

#### Step 1：开篇定调（100-150 字）

简短一段，定下今年的基调：

- 用术语描述命局对火/土/金/水/木的喜忌（一句话）
- 流年的核心特征是什么（一句话）
- 流年与命局的整体共振情况：补命局所缺，还是加重偏势？是顺势还是逆势？
- 用一句话概括今年是什么样性质的一年（不评判好坏，描述能量类型）

**示例措辞：**
> "今年丙午年，纯火之年——天干丙火、地支午火藏丁火，对您水势极旺、火金不显的命局来说，是喜用神到位的一年。"

---

#### Step 2：作用关系表（核心，必须有）

这是这一节质感最关键的部分。用 markdown 表格列出**流年与原局四柱、当前大运**的所有关键作用关系。表格三列：

| 作用关系 | 具体组合 | 解读 |
|---|---|---|
| [作用名称] | [流年干支 + 原局/大运干支] | [一句话生活化解读] |

**必须包含的作用类型：**
- 流年天干对日主的十神性质（如"丙火正财透出"）
- 流年地支与原局四支的合冲刑害关系（每条单独一行）
- 流年与当前大运的作用关系
- 流年与日支（婚姻宫）的作用——单独标注，因为这个特别重要
- 流年与时支（子女宫/晚年宫）的作用

**每条解读要做到：**
- 命理结构 → 生活化结果的一句话翻译
- 不下断言（不说"会发生 X"）
- 用倾向性语言（"有 X 的倾向""倾向于 X"）

**示例（参考这个质感）：**

| 作用关系 | 具体组合 | 解读 |
|---|---|---|
| 天干正财 | 丙火透出 | 正财外显，今年是收入和回报相对清晰的一年 |
| 寅午半合 | 时支寅 + 流年午 | 半合火局，命局里偏弱的火得到激活，做事的"火力"比平常足 |
| 午未六合 | 流年午 + 大运未 | 大运与流年紧密结合，外部环境对命主有支持 |
| 丑午相害 | 日支丑 + 流年午 | 日支被害，是值得在亲密关系上多花心思的一年 |

表格之后，用一句话总结整体作用关系的特征：合多冲少 / 喜神到位 / 用神受冲 / 等等。

---

#### Step 3：分面向解读（每个面向 100-200 字）

按以下顺序展开（如果某个面向命局信息很弱可以一笔带过）：

**事业与工作**
- 基于流年正官/七杀/食伤/财星等十神的状态
- **结合命主真实身份做具体化**（如果用户提供了职业信息）
- 倾向性表达：今年的事业方向上倾向于 X，X 方面有机会出现 X
- 季节性提示：哪几个月份（用农历月）能量最为集中

**财运**
- 流年财星的状态（透干？藏支？被冲？）
- 财气的形态特征（稳定 vs 流动、正财 vs 偏财）
- 需要警惕的财务关注点（如果有比劫旺、流年破财库等结构）

**感情/婚姻**
- 流年与日支（婚姻宫）的具体作用
- 已婚或未婚的不同重点
- 如果有冲、害、刑等张力性作用，**坦诚标注但不夸大**——参考 Step 2 示例里"丑午穿"那种处理：标出隐患、说明特征、给出应对、不夸张
- 措辞要点：用"值得多花心思""可能出现 X 类型的小摩擦"，不用"会出问题""有危机"

**健康**
- 命局原本五行偏弱对应的器官系统（不变的部分）
- 流年五行变化带来的健康影响（补益还是加重）
- 季节性提示（如夏天/冬天哪个月份特别要注意什么）

---

#### Step 4：全年节奏建议表（必须有）

用表格列出全年的节奏建议，让命主有可执行的时间维度参考：

| 时段 | 月份（农历） | 关键词 | 建议 |
|---|---|---|---|
| 上半年 | 正月-四月 | [关键词] | [一句话建议] |
| 高峰期 | [流年喜用月份] | [关键词] | [一句话建议] |
| 下半年 | [对应月份] | [关键词] | [一句话建议] |
| 收尾期 | 十一月-腊月 | [关键词] | [一句话建议] |

**关键词举例：** "蓄势待发""全面发力""落地收获""总结复盘"等

**建议要做到：**
- 可执行（不是"加油""努力"这种空话）
- 与流年喜忌一致（喜用月份建议主动，忌神月份建议守成）
- 不预言具体事件，只给方向

---

#### Step 5：总结段（150-250 字）

最后写一段连贯的总结：

- 一两句话概括今年的整体性质
- 重申今年最值得发力的方向
- 重申今年最值得留意的隐患
- 一个建设性的提醒（不是鸡汤）
- **可选：** 一句命理诗化的收尾（如"癸水汪洋逢丙照，午寅半合暖冬潮"这种）。**如果不擅长写命理诗，宁可不写也不要勉强凑——拙劣的诗会拉低整段质感。**

### 重要的措辞约束

**不要用的措辞**（即使凭直觉很想写）：

| 错误措辞 | 问题 |
|---|---|
| "近年来运势最为强劲的一年" | 评判式判断 |
| "财星当头，收入增长、事业回报显著" | 具体事件预测 |
| "有获得更高层级授权或管理范围扩大的可能" | 过于具体的职位预测 |
| "身强指数 62.8" | 伪精准数字（命理学没有公认的此类指标） |

**要用的措辞：**
- "今年是命局结构上喜用神到位的一年，能量上倾向于支持发力型行为"
- "正财透出意味着收入和回报的可见度比以往清晰，倾向于通过劳动转化为有形成果"
- "今年在职位、授权、管理范围上有出现变化的命理结构，但具体方向需要看个人选择"

**核心原则：** 把"会怎样"改成"倾向于怎样"或"有 X 类型的能量"。把"显著""强劲"这种评判词换成中性描述。

### 结合命主身份的具体化（需要克制）

如果用户在提示里给了命主的职业、婚姻状态等信息，可以**适度结合**做具体化解读。但要注意：

- 结合身份是为了让分析更贴合，不是做职业预测
- 说"基于您的工作性质，这种能量倾向于在 X 方面发力"，而不是"您今年会拿到 X 项目"
- **如果用户没提供身份信息，不要主动猜测命主从事什么职业**——基于命局推断职业是另一个层面的判断，容易出错
