# 各章节详细格式

报告共 7 章：

1. 排盘信息（脚本输出，原样放入）
2. 整体命局观察（命理观察层）
3. 生活面向分析（按年龄段展开各面向）
4. 当前大运分析
5. 下一大运分析
6. 整体画像与人生主线（散文，命主直接读）
7. 今年的运势分析（落到当下）

**没有"重要年份与人生节点"章节**——具体年份判断只在脚本输出的"流年作用关系完整表"里作为内部参考，对外报告**不列出**（这是工具的核心定位，详见 SKILL.md）。

---

## 二、整体命局观察

### 命局核心特征（术语层，给命理师看）
列点输出：
- 格局判断（取格名称 + 简要说明格的结构）
- 用神喜忌（主用神 + 辅用神，标注是否在原局得力）
- 命局核心张力（核心的生克关系，一两句话）
- 缺失维度（明字缺的五行 / 藏而不透的十神 / 被冲合的关键位置）
- 命局亮点（特别有力的结构）
- **真神判断**（来自 methodology Step 7-1，月令藏干 + 时支藏干）
- 特殊结构（伏吟、反吟、天克地冲、双合等，如有才写）

### 人格画像（三视角融合）
3-5 段，每段聚焦一个核心特征，每段后标注【稳】【分】或【待】。

**融合三视角写：**
- 子平视角的命理判断是"骨"
- 盲派视角的象意和画面是"肉"
- 现代心理学翻译是"翻译层"，让判断更贴近人的实际体感

**注意：** 这一节是给命理师看的判断层，不需要单独写通俗表达。给命主的通俗整合在第六节完成。

## 三、生活面向分析

### 按年龄段选择展开面向

| 年龄段 | 展开面向 |
|---|---|
| 18 岁以下 | 学业、性格、家庭关系、潜在天赋 |
| 18-28 岁 | 学业收尾、工作起步、性格、感情萌芽、当前大运 |
| 28-40 岁 | 工作发展、婚姻、子女（如已婚有子女）、财运、当前+下一大运 |
| 40-55 岁 | 工作转型、婚姻状态、子女、财运、健康、大运转换 |
| 55 岁以上 | 人生整合、健康、子女、晚年方向、财富守成 |

### 每个面向的格式

```
### [面向名称]
**命理观察**：[术语+判断。明确标注透干还是藏干]
**通俗表达**：[100-200 字连贯段落，大白话，第二人称"你"，不用列表]
[确定性标注]【稳】【分】或【待】
```

### ⚠️ 财运面向特别要求

财运面向**必须走完 methodology.md 的 Step 7（盲派做功效率分析）**：
- 真神状态（被冲坏？完好？）
- 家内 vs 家外的财官归属
- 是否有"财过河"结构
- 做功效率评估（高效 / 中效 / 低效 / 不做功）
- 三宝配置（印 / 食 / 禄）
- 是否容易被流年比劫引动

只用传统派"财星有力无力"做财运分析颗粒度不够。

## 四、当前大运分析

### 大运基本信息（从排盘脚本输出取）
- 干支、起止年份、已走年数、剩余年数
- 大运天干十神性质
- 与原局的作用关系（**直接读脚本输出的"各大运与原局的作用关系"表，不要手算**）

### 这十年的能量主题

**给命理师的版本**：用术语描述命理结构和作用关系

**通俗版本**：100-200 字连贯段落，大白话描述这十年是什么样的阶段、有什么能量主线、命主可能的体感是什么。不用术语，不用列表，流畅段落。

### 这个阶段的"贵人"方向
不用宿命式判断，用关系式建议：
- 命局缺什么 + 当前大运需要什么 → 能补这个的人就是贵人
- 描述贵人的"特征类型"而非"生肖/方位"

### 这个阶段需要警惕的消耗模式
觉察式建议，不是禁令：
- 这个大运容易透支什么
- 命局原本的弱点在这个运里是否被放大
- 容易陷入的关系/财务/健康模式

## 五、下一大运分析

与当前大运对比（延续？反向？过渡？）

## 六、整体画像与人生主线（最重要）

这是整份报告里**唯一会被命主完整读到的部分**。600-800 字散文，老派命理师的口吻——结构清楚，面向齐全，每个面向一两句话说透，整体精炼有力。

### 写作的核心模型

想象一位经验丰富的老命理师坐在你对面，看完你的盘后用 600-800 字告诉你这一生的全貌。他的语言不文学化、不堆比喻、不反复铺陈。他**按面向逐一说**：你是个什么样的人，你适合做什么事、不适合做什么事，你的财运怎么走，你的婚姻怎么样，你的子女怎么样，你的健康要注意什么，你这辈子的关键节点在哪里。

每个面向**只说一两句到位的话就过**，不在一个面向上反复打转。

### 必须按顺序覆盖的 7 个面向 + 1 句收尾

1. **性格底色**（约 80-150 字）：这个人最核心的一两个特征
2. **适合什么/不适合什么**（约 80-150 字）：发力方向、避开方向
3. **财运**（约 60-120 字）：钱怎么来、什么样的财、需要警惕什么
4. **婚姻/亲密关系**（约 60-120 字）：在关系里的模式、配偶大概什么样、关系张力点
5. **子女/与下一代的关系**（约 40-80 字）：缘分如何、关系模式
6. **健康**（约 40-80 字）：要留意的身体面向
7. **大运节点/人生主线**（约 100-150 字）：当前+下一大运，一生大致脉络

**收尾**（一两句话）：一句真观察，不是鸡汤

**总长度控制在 600-800 字**。如果某个面向命局信息很弱，那一段就少写或一笔带过——不要为了凑字数硬编。

### ⚠️ 涉及人生阶段时：用节奏感，不用具体年份

工具的核心定位是"理解整体格局，不预测具体年份"。所以在第六节整体画像里，**所有涉及人生时间的描述，必须用节奏感和阶段感的表达，不要给出具体年份**。

**禁用**（即使脚本算出了高危信号）：
- "2024 年婚姻有危机"
- "2027 年事业转折"
- "X 岁那年要注意健康"

**允许**：
- "中年这一段在感情上会经历一次明显的调节，过去了关系会更稳"
- "事业上有过一次大的转折，已经过去；接下来 10 年是把这次转折真正消化、找到下一段方向的时间"
- "身体上需要留意的是脾胃和睡眠，特别是 50 岁前后"（用大致年龄段，不用具体年份）
- "你这辈子在感情上一直需要主动经营，不是那种'凑合就过'的命局"

**节奏感的语言库**：
- "中年这一段..."
- "进入下半生..."
- "过去这十年..."
- "接下来这十年..."
- "X 岁前后这几年..."（5 年级别的窗口，不是具体某年）
- "下一个大运里有..."（用大运标记而不是公历年）

**为什么用节奏感**：
- 精确年份的判断不可靠（算法可能错），错了就动摇命主的信任
- 命主对"我应该怎样过这辈子"的认知，节奏感比年份精确度有用得多
- 节奏感的表达对命主有**持久价值**——他知道整体趋势，可以自己应用到任何具体年份

**内部参考与对外输出的关系**：

你从脚本输出的"流年作用关系完整表"里**看到了具体年份和关键信号**——这些信息是给你做内部推理用的，让你的"节奏感"判断有数据支撑。但**对外报告里不要把具体年份列出来**。

例如：脚本告诉你 38 岁有比劫剋财信号，50 岁有伏吟+财弱信号——你的内部理解是"这个命主在感情上有规律性的张力"，对外表达成"你这辈子在感情上一直需要主动经营，每隔一段时间都会有需要重新调节的时刻"。

### ⚠️ 人格驱动力调性必须体现

整体画像必须体现命主的"主动型 vs 被动型"人格驱动力（来自 methodology.md 元规则二）——搞错驱动力类型，整段画像都会偏。

- **主动型命主**的调性：聚焦"你的选择""你的判断""你想做的事"
- **被动型命主**的调性：体现"环境塑形 + 你的回应方式"的双向感，但**不要使用"被推着""被命运""被摆布"等宿命感词汇**——用"这些机会找上你""这种关系塑造了你的轨迹""你在回应外部信号的过程中长出来的"等中性表达
- **被动型命主**的"事业方向、换城市、重大转变"等节点，可以适度提示"这种变化常常和你身边的关系/责任有关"

### 写作技法（7 条）

#### 第一：每个面向只用一两句话说透，不反复铺陈

老派命理师的力量在"一句话定调"。不要在一个面向里写三段话来反复表达同一件事。**说到了，就过。**

#### 第二：话题转换要清晰，不要把所有面向揉成一锅粥

读者要能明显看出"现在开始讲财运了""现在转到婚姻了"。用承接句切换：

- "再说工作和事业。"
- "财运这一块。"
- "婚姻和感情上。"
- "子女方面。"
- "身体上要留意的是。"
- "再讲讲你这辈子的大运节点。"

不要用 markdown 小标题（不能用 `###` 或 `**财运**` 这种）。

#### 第三：避免文学化堆砌

不要用比喻堆砌（"像河流一样""像山一样""像火焰一样"），不要用排比铺陈（"你 X，你 Y，你 Z"），不要用形容词堆叠（"深邃的、敏锐的、富有创造力的"）。

**老命理师不写散文**。他说的是结论，不是抒情。

#### 第四：避免命理术语和命理黑话

不能用：伤官、七杀、比劫、用神、化解、显化、引动、应事、信息、结成、透出。这些命主听不懂。

可以用：大运、流年、八字、命局、五行（这些入门级词汇命主基本都能理解）。

#### 第五：敢说硬的部分，但用建设性的方式

如果某个面向有真实的张力（比如婚姻有冲、健康有压力），**直说**，但说完要给一个怎么应对的方向。

例子：
- 不写："你婚姻不好。"
- 也不写："你的婚姻完全没问题。"
- 写："婚姻上有个特点——你的伴侣大概是个有自己主张、不太顺着你的人。这段关系里有张力，但这种张力本身不是坏事，关键是你怎么用它。"

#### 第六：结尾用一句真观察收，不要鸡汤

禁用结尾："愿你..."、"祝你..."、"未来可期"、"加油"。

可以这样结尾：
- "这是你这辈子的一个核心议题，不是一个阶段的事。"
- "走好这十年，下一个阶段你会更舒展。"
- "你现在做的事情，是为五年后铺路。"

#### 第七：避免金句复用和同质化表达（最重要）

详细的禁用金句列表见 `style_guide.md`。

**核心原则**：每次写画像时，**强制基于这个具体盘的具体结构**来构造表达。

不是"这个人内向 → 套用'想得多说得少'"，而是先问："这个人的内向具体来自哪里？"
- 印星深透 → 思考型内向
- 日支孤辰 → 社交型疏离
- 食伤被克 → 表达受阻
- 比劫弱 → 不愿群体取暖

不同来源对应的内向是不一样的——**用不一样的话来描述**。

**自检：** 写完画像后回看，问"这段话里有没有任何一句，是我能直接搬到另一个相似命主身上也能成立的？"如果有，那一句就需要重写——让它**只对这个特定盘成立**。

### 范文骨架示例

⚠️ **下面这个示例只是用来说明结构和节奏**——7 个面向怎么排、承接句怎么用、长度怎么控制。**不要照搬里面的具体句子**，每次都要根据真实命局重新构造表达。

```
[第一段：性格底色，80-150字]
[切入语 + 一两句核心特征 + 一句这个特征的优势/陷阱]

[第二段：适合什么 / 不适合什么，80-150字]
"适合做的事..." 或 "事业上..." 切入
[发力方向 + 避开方向 + 一句对你独特工作模式的描述]

[第三段：财运，60-120字]
"财运这一块..." 或 "钱财上..." 切入
[财气从哪里来 + 求财方式 + 需要警惕什么]

[第四段：婚姻，60-120字]
"婚姻上..." 或 "感情这一块..." 切入
[配偶的大概特征 + 关系里的张力点 + 一句建设性的提醒]

[第五段：子女，40-80字]
"子女方面..." 切入
[子女缘分的形态 + 与下一代关系模式]

[第六段：健康，40-80字]
"身体上..." 切入
[最需要留意的器官/系统 + 容易透支的方式]

[第七段：大运节点和人生主线，100-150字]
"再说说你这辈子的节点..." 切入
[过去 → 当前 → 未来的脉络 + 关键转换点。用节奏感，不用具体年份]

[结尾一句]
[真观察，不是鸡汤]
```

**这个骨架是死的，每次填进去的内容必须是活的——基于这个具体盘的具体结构构造。**

## 七、今年的运势分析

这一节专门分析当前流年（脚本输出的"当前流年"）对命主的影响。命主最关心的就是"今年怎么样"——这一节要讲清楚。

**为什么只有这一节例外能涉及具体年份**：因为这是命主当下能感受、能行动的时间窗口。其他章节（特别是第六节）涉及人生阶段都用节奏感表达，但今年是真实在发生的事。

### 整体输出要求（最关键的质感原则）

**用专业但不堆砌术语的方式写**——每出现一个命理术语（如"正财透干""寅午半合""午未六合"），后面立刻配一个生活化的解读（如"财气浮上水面""喜用力量倍增""贵人相助"）。

**这是这一节质感最重要的来源。** 命理结构是骨，生活化解读是肉，两层缺一不可。只有结构没有解读 → 像八股；只有解读没有结构 → 像玄学；两者并列 → 才是专业又可读的命理报告。

### 标准输出结构（按顺序）

#### Step 1：开篇定调（100-150 字）

简短一段，定下今年的基调：

- 用术语描述命局对各五行的喜忌（一句话）
- 流年的核心特征（一句话）
- 流年与命局的整体共振：补命局所缺还是加重偏势？顺势还是逆势？
- 一句话概括今年是什么性质的一年（描述能量类型，不评判好坏）

**示例措辞：**
> "今年丙午年，纯火之年——天干丙火、地支午火藏丁火，对您水势极旺、火金不显的命局来说，是喜用神到位的一年。"

#### Step 2：作用关系表（核心，必须有）

这是这一节质感最关键的部分。用 markdown 表格列出**流年与原局四柱、当前大运**的所有关键作用关系——这些**直接从脚本输出的"流年作用关系完整表"里取当年那一行**，不要手算。

| 作用关系 | 具体组合 | 解读 |
|---|---|---|
| [作用名称] | [流年干支 + 原局/大运干支] | [一句话生活化解读] |

**必须包含的作用类型：**
- 流年天干对日主的十神性质（如"丙火正财透出"）
- 流年地支与原局四支的合冲刑害关系（每条单独一行）
- 流年与当前大运的作用关系
- 流年与日支（婚姻宫）的作用——单独标注，因为这个特别重要
- 流年与时支（子女宫/晚年宫）的作用

**每条解读要做到：**
- 命理结构 → 生活化结果的一句话翻译
- 不下断言（不说"会发生 X"）
- 用倾向性语言（"有 X 的倾向""倾向于 X"）

**示例（参考这个质感）：**

| 作用关系 | 具体组合 | 解读 |
|---|---|---|
| 天干正财 | 丙火透出 | 正财外显，今年是收入和回报相对清晰的一年 |
| 寅午半合 | 时支寅 + 流年午 | 半合火局，命局里偏弱的火得到激活，做事的"火力"比平常足 |
| 午未六合 | 流年午 + 大运未 | 大运与流年紧密结合，外部环境对命主有支持 |
| 丑午相害 | 日支丑 + 流年午 | 日支被害，是值得在亲密关系上多花心思的一年 |

表格之后，用一句话总结整体作用关系的特征：合多冲少 / 喜神到位 / 用神受冲 / 等等。

#### Step 3：分面向解读（每个面向 100-200 字）

按以下顺序展开（如果某个面向命局信息很弱可以一笔带过）：

**事业与工作**
- 基于流年正官/七杀/食伤/财星等十神的状态
- **结合命主真实身份做具体化**（如果用户提供了职业信息）
- 倾向性表达：今年的事业方向上倾向于 X，X 方面有机会出现 X
- 季节性提示：哪几个月份（用农历月）能量最为集中

**财运**（**必须用 Step 7 盲派做功方法**）
- 流年财星的状态（透干？藏支？被冲？）
- 流年是否引动比劫（破财应期信号，必须扫描）
- 财气的形态特征（稳定 vs 流动、正财 vs 偏财）
- 流年对真神的作用（是助是损？）
- 需要警惕的财务关注点（如果有比劫旺、流年破财库等结构）

**感情/婚姻**
- 流年与日支（婚姻宫）的具体作用
- 已婚或未婚的不同重点
- 如果有冲、害、刑等张力性作用，**坦诚标注但不夸大**——参考 Step 2 示例里"丑午穿"那种处理：标出隐患、说明特征、给出应对、不夸张
- 措辞要点：用"值得多花心思""可能出现 X 类型的小摩擦"，不用"会出问题""有危机"

**健康**
- 命局原本五行偏弱对应的器官系统（不变的部分）
- 流年五行变化带来的健康影响（补益还是加重）
- 季节性提示（如夏天/冬天哪个月份特别要注意什么）

#### Step 4：全年节奏建议表（必须有）

| 时段 | 月份（农历） | 关键词 | 建议 |
|---|---|---|---|
| 上半年 | 正月-四月 | [关键词] | [一句话可执行建议] |
| 高峰期 | [流年喜用月份] | [关键词] | [一句话] |
| 下半年 | [对应月份] | [关键词] | [一句话] |
| 收尾期 | 十一月-腊月 | [关键词] | [一句话] |

**关键词举例：** "蓄势待发""全面发力""落地收获""总结复盘"等

**建议要做到：**
- 可执行（不是"加油""努力"这种空话）
- 与流年喜忌一致（喜用月份建议主动，忌神月份建议守成）
- 不预言具体事件，只给方向

#### Step 5：总结段（150-250 字）

最后写一段连贯的总结：

- 一两句概括今年整体性质
- 重申最值得发力的方向
- 重申最值得留意的隐患
- 一个建设性提醒（不是鸡汤）
- **可选：** 一句命理诗化的收尾。**如果不擅长写命理诗，宁可不写也不要勉强凑——拙劣的诗会拉低整段质感。**

### 重要的措辞约束

**不要用的措辞**（即使凭直觉很想写）：

| 错误措辞 | 问题 |
|---|---|
| "近年来运势最为强劲的一年" | 评判式判断 |
| "财星当头，收入增长、事业回报显著" | 具体事件预测 |
| "有获得更高层级授权或管理范围扩大的可能" | 过于具体的职位预测 |
| "身强指数 62.8" | 伪精准数字 |

**要用的措辞：**
- "今年是命局结构上喜用神到位的一年，能量上倾向于支持发力型行为"
- "正财透出意味着收入和回报的可见度比以往清晰，倾向于通过劳动转化为有形成果"

**核心原则：** 把"会怎样"改成"倾向于怎样"或"有 X 类型的能量"。把"显著""强劲"这种评判词换成中性描述。

### 结合命主身份的具体化（需要克制）

如果用户在提示里给了命主的职业、婚姻状态等信息，可以**适度结合**做具体化解读：

- 结合身份是为了让分析更贴合，不是做职业预测
- 说"基于您的工作性质，这种能量倾向于在 X 方面发力"，而不是"您今年会拿到 X 项目"
- **如果用户没提供身份信息，不要主动猜测命主从事什么职业**
