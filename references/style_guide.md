# 样式规范

## 命理黑话禁用清单

这类词对内行是日常用语，对外行是黑话——既不像纯术语那样硬核，也不是大白话。写"通俗表达"和第六节"整体画像"时必须换掉。

| 禁用词 | 改成什么 |
|---|---|
| 子女信息 | 子女缘分 / 与子女的关系 |
| 显化 / 显象 | 出现 / 落实 / 真正发生 |
| 引动 | 触发 / 激活 |
| 化解 | 缓解 / 转化 |
| 结成 / 成局 | 形成 / 落实 |
| 应事 / 应期 | 对应的实际事件 / 对应的时间 |
| 主...（如"主婚姻晚成"） | 对应着... / 这意味着... |
| 用事 | 起作用 |
| 透出 | 在天干明显出现 |
| 受制 | 被压制 / 被克制 |
| 透情 | 表现出来 |
| 见象 | 看到对应的事 |
| 喜... 忌... | 这个人需要... / 这个人需要避免... |

**原则：** 命理术语（十神名、地支作用、格局名、五行关系）在给命理师看的那层是 OK 的——伤官、七杀、比劫、刑冲合害、用神等可以直接用。但"那些介于术语和大白话之间的行话"必须换成大白话。

**自检方法：** 写完命理观察那段后，问"如果我把这段念给一个不懂命理但中文很好的朋友听，他能听懂吗？"专业术语他听不懂没关系，但黑话词他会愣住——那就要换掉。

### 反面例子 vs 正面改写

**反面例子（容易出现的写法）：**

> "时柱甲寅虽为子女宫，但寅中藏戊（正官）—— 子女信息在时柱有，但偏弱"

问题：
- "子女信息"是黑话词，外行看不懂
- "在时柱有，但偏弱"——"有"什么？"什么"偏弱？指代不清

**正面改写：**

> "时柱甲寅是子女宫，寅中藏戊土（藏干正官，男命以正官代表子女）——子女星在时柱位置上有，但只是藏干不透，力量偏弱。这意味着子女缘分的体现可能比较平和，不是命局里特别强的主题。"

这样写既保留了专业判断（"藏干""不透""力量偏弱"），又把黑话词翻译成了大白话（"子女信息"→"子女星""子女缘分"），整段也有了清晰的因果链。

## 第六节"整体画像"写作规范

### 话题转换用承接句（禁用 markdown 小标题）

正确做法（用这类句子切换面向）：
- "再说工作和事业。"
- "财运这一块。"
- "婚姻和感情上。"
- "子女方面。"
- "身体上要留意的是。"
- "再讲讲你这辈子的大运节点。"

禁止用 `###` 或 `**财运**` 这种 markdown 标题切换。

### 禁用文学化堆砌

不要用比喻堆砌（"像河流一样""像山一样""像火焰一样"）；不要用排比铺陈（"你 X，你 Y，你 Z"）；不要用形容词堆叠（"深邃的、敏锐的、富有创造力的"）。

老命理师不写散文，他说的是结论，不是抒情。

### 结尾禁用语
禁止："愿你..."、"祝你..."、"未来可期"、"加油"

可以这样结尾："这是你这辈子的一个核心议题，不是一个阶段的事。" / "走好这十年，下一个阶段你会更舒展。" / "你现在做的事情，是为五年后铺路。"

### ⚠️ 涉及人生时间的描述必须用节奏感

详见 `output_structure.md` 第六节"涉及人生阶段时：用节奏感，不用具体年份"。**第六节绝对不能出现具体公历年份**，即使脚本算出了高危信号。

### ⚠️ 避免"前 X 年""后 X 年"造成年龄歧义

**问题**：命理师习惯用"前 X 年"指代**大运周期**（从起运算），但命主会从**自然年龄**理解——这两者经常有 10 年左右的偏差（起运岁数差异），会让命主困惑甚至怀疑分析的准确性。

**典型反面例子**："你的前三十年都在积累" — 命主已经活了 40 年了，会想"那前十年呢？"

**正面改写方式（三选一）**：

1. **明确岁数区间**："从你 10 岁起运到 39 岁这三步大运（共 30 年）是积累期"
2. **用大运步数命名**："起运后的前三步大运是积累期"
3. **加一句解释起运时间**："你 10 岁才起运，所以从命运层面真正'开始算'是从 10 岁——10 到 39 岁这三步大运是积累期"

**核心原则**：

- 描述大运阶段时，**必须明确起运岁数 + 具体岁数区间**，不能简单说"前 X 年"
- 如果命主起运较晚（如 8 岁、10 岁起运），**第一次提到"前几年"时必须解释清楚"前几年"指的是大运周期，不是自然年龄**
- 童年（起运前）的描述要单独处理，明示"起运前是原生家庭塑造期，不算入大运积累阶段"

**为什么这条重要**：命主对命理的信任建立在"数据一致性"上——如果"前三十年"和"我已经活了 40 年"对不上，命主会本能地怀疑整套分析的准确性。这种困惑会扩散到整份报告。

## 高频金句禁用列表（最重要）

不同命主的画像里不能反复出现这些"听起来很准"的同质化金句：

**禁用：**
- "想得多说得少" / "脑子比嘴快" / "想得多做得少"
- "妈妈的稳定就是你的稳定" / "父亲是你的贵人" 等亲属关系金句
- "你身上有一股劲" / "你是流动的" / "你是火焰一样的人" 等比喻型定调
- "你需要的是..." / "你这辈子最该练的..." 等总结性金句
- "扛事最多的十年" / "被推到不得不扛的位置上" / "被压着烧"
- "你不是孤狼型的人" / "你是要在群里才会发光的"
- "事情'成'的窗口期" / "提前布局" / "为五年后铺路"
- "想和做的平衡" / "智慧够了差的是行动力"
- 任何引号包起来的"金句式"短语（"成"、"扛"、"松"等带引号强调的字）

### 正确做法：每次描述都基于具体命局结构

不是"这个人内向 → 套用'想得多说得少'"，而是先问：这个人的内向**具体来自哪里**？

| 来源 | 对应的不同表达 |
|---|---|
| 印星深透（思考型内向） | "你是那种凡事要自己想清楚才动的人。" |
| 日支孤辰（社交型疏离） | "你不是天生爱热闹的人，独处对你不消耗反而充电。" |
| 食伤被克（表达受阻） | "你心里想的东西比说出来的多得多，有时是不愿说，有时是说不出来。" |
| 比劫弱（不愿群体） | "朋友圈不是你的能量来源，你不靠人多取暖。" |

**不要用通用模板套不同的人。** 每个人的命局结构都不一样，画像里的具体表达就应该不一样。

**自检方法：** 写完画像后回看，问"这段话里有没有任何一句，是我能直接搬到另一个内向型命主身上也能成立的？"如果有，那一句需要重写——让它**只对这个特定盘成立**。

## 用真实经历校准

如果提供了命主的真实经历（学业、职业、婚姻状态等），用这些经历来校准判断：

- 已被经历验证的命局解读 → 高确定性，直接写
- 暂未被经历验证的判断 → 标注【待】
- 与经历明显冲突的判断 → 标注【与实际不符，需思考是否是命理解读错误或经历有特殊原因】

## 女命处理原则

传统命理对女命有过时的性别预设。分析女命时：
- 不使用"克夫""旺夫"这类语言
- 不基于性别预设事业上限
- 财官星的解读对男女一视同仁
- 婚姻分析聚焦"关系模式"而非"婚姻成败"
- 子女分析不预设女性必须生育

---

## 亲密关系问题处理

如果命主问"我老公/老婆/对象怎么样"或"我们合不合"：
- 不评判第三方的命格
- 只分析命主自己在亲密关系中的模式
- 建议聚焦在"你可以怎么调整自己""你可以怎么看待这段关系"

---

## 敏感话题（健康、重大风险）

- 在第八节"今年的运势分析"里诚实标注命主当下需要特别留意的方向
- 在第六节"整体画像"里用**节奏感**表达健康关注（"中年这一段身体上需要主动调节，特别是 X 系统"），不给具体年份
- 绝对不直接预测疾病或意外（不用"会得 X 病""会出 X 事"）
- 用觉察式表达："这个阶段身体的可持续性需要主动管理" / "压力承受需要主动管理" / "值得留意的窗口期"
- **即使脚本算出某些年份是高危信号，对外报告也用"X 岁前后这几年"这种 5 年级别的窗口，不用具体年份**

**人机分工的原则：** 命理师（用户）的判断比命理工具的判断更重要——用户会决定哪些话适合在当下场景说出口。所以这个 skill 的输出是"原料"，不是"成品"。把分析做扎实、把张力点诚实标注，把语气控制在中性，让用户自己挑该说和不该说的。

---

## 盲派威胁性判语（绝对禁用）

盲派理论里有一些威胁性判语，**全部不允许使用**——即使内部推理走到了这里，对外也必须翻译成中性描述：

| 禁用判语 | 该换成什么 |
|---|---|
| 克夫 / 克妻 / 克兄弟 / 克父母 | 描述"关系里的张力""相处模式"，不评判"克" |
| 行、吃、急、予、瘫（五种残疾） | 绝对不与命局结构挂钩——这是命理学里最不可靠的部分 |
| 贫命 / 乞丐 | "财气结成方式比较间接""需要通过持续做事来积累" |
| 巨富 / 大富 | "财气结成的效率较高""命局结构对财富形成是支持的" |
| 刑伤 / 灾煞应期 | "X 阶段需要在安全方面更主动""能量结构上有动荡的特征" |
| 任何"命局 → 具体灾难/残疾/重大事件"的直接挂钩 | 不输出 |

**原则**：盲派的**方法论**有价值（做功效率、家内家外、真神理论、引动比劫等），但**威胁性判语**是命理学里需要被剥离的部分——它们不增加判断的准确性，只增加命主的焦虑。

---

## 过度自信的层级定盘（禁用）

不说类似下面这种**具体职级、量级**的判断：

- "处级及以上""副厅级"——具体职级禁用
- "年薪 X 百万级别""资产千万级"——具体量级禁用
- "财富层级在 X 阶段会突破到 X 级别"——伪精准描述禁用

可以说的：
- "命局结构对事业承担更大平台是支持的"
- "财富形成的方式比较稳定，整体趋势是积累型而非爆发型"
- "做事的格局倾向于在更大范围内被看见"
