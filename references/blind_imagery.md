# 盲派象意参考表（事件预测弹药库）

这份表是**预测定位下**的核心弹药库——盲派预测具体事件靠的是"象意"（每个十神、宫位、刑冲合害对应的现实画面）。子平判旺衰用神是"骨"，象意是"肉"，没有象意，预测引擎只能说"今年财运一般"这种无信息量的话。

**三个用途：**
1. **首盘演绎**：把命局结构翻译成可验证的具体画面（"早年家中长辈对你影响重""某段大运有过事业方向大转"），铺反馈面。
2. **流年/大运应事**：把脚本算出的作用关系（冲、合、刑、害、十神透干）翻译成"可能应在什么事"。
3. **信号↔面向反查**（见文末表）：命主反馈某年发生过某类事 → 反查该年应该出现什么信号 → 用于时辰反推（`bazi_chart.py --verify-events`）和解读分支收敛。

**所有象意输出对外时必须过一遍 [style_guide.md](style_guide.md) 的措辞红线**——负面象意只能用觉察式/中性表达，威胁性判语（克夫/克妻/伤灾/贫命等）一律禁用。本表"负面象"列是给命理师内部判断用的原料，不是对外措辞。

---

## 一、十神象意

每个十神带三层：**人物**（应在谁）/ **事物**（应在什么）/ **事件画面**（动起来时发生什么）。正负两面都列——同一个十神，真神/坐实时偏正面，虚透/受制时偏负面（真假判定见 methodology Step 7-7）。

### 比肩
- **人物**：兄弟姐妹、同辈、同事、同行、合伙人、自己的分身
- **事物**：自我、独立、本钱、体力、竞争场域
- **正面象**：得朋友之助、合伙成事、独立创业、靠自己打拼站稳
- **负面象**：争夺、分家分财、竞争耗损、朋友拖累、各自为政

### 劫财
- **人物**：异性兄弟、合伙人、对手、表面的朋友
- **事物**：投机、赌性、本钱外流的通道
- **正面象**：仗义、行动力强、敢拼、关键时有人搭手
- **负面象**：破财、被借钱不还、合伙被坑、过河拆桥、命犯小人（虚透时尤甚）

### 食神
- **人物**：晚辈、学生、下属；女命的子女（女儿偏多）
- **事物**：才华、口才、福气、饮食、艺术、产品、技术输出
- **正面象**：才华变现、专业产出、生活有滋味、子女缘（女命）
- **负面象**：过于安逸、表达散漫、被枭印夺食（才华无法落地）

### 伤官
- **人物**：晚辈、学生；女命的子女（儿子偏多）
- **事物**：才艺、表达欲、技术、叛逆、锋芒、批判力
- **正面象**：才华出众、技术突破、靠本事出名、不走寻常路成事
- **负面象**：恃才傲物、口舌是非、与权威冲突、伤官见官（事业/婚姻动荡）、伤灾（**对外只能说"安全方面需主动留意"**）

### 正财
- **人物**：妻子（男命）、稳定的合作方
- **事物**：固定收入、薪水、节俭积累的钱、可掌控的资源
- **正面象**：稳定进财、靠劳动得财、婚姻落实（男命）、持家
- **负面象**：财来财去守不住、为钱所困、被财牵绊、婚姻压力（男命财星受冲）

### 偏财
- **人物**：父亲、情人/异性缘（男命）、客户、流动的合作方
- **事物**：横财、外快、流动资金、生意、众人之财、机会财
- **正面象**：财路宽、生意机会、慷慨、人脉变现、偏财运
- **负面象**：投机失利、财务波动大、因财生情/因情破财（男命）、父缘事件

### 正官
- **人物**：丈夫（女命）、上司、政府、权威、规则制定者
- **事物**：工作、职位、名誉、约束、地位、考试功名
- **正面象**：升职、得名、工作稳定、婚姻落实（女命）、被认可
- **负面象**：压力、被管束、官非、名誉受损、工作变动（正官受冲合）

### 七杀
- **人物**：情人/有压迫感的对象（女命）、竞争对手、严厉的上司、小人
- **事物**：压力、竞争、责任、权力、风险、疾病、突发
- **正面象**：担大任、独当一面、化压力为权力（七杀有制）、魄力、危机中崛起
- **负面象**：被倒逼、健康透支、竞争激烈、小人、惊险事件（**对外只能说"压力承受需主动管理""动荡特征"**）

### 正印
- **人物**：母亲、师长、贵人、保护者
- **事物**：学历、文凭、证书、房产、靠山、名声、保障、宗教信仰
- **正面象**：升学、得贵人、买房、获保护、名声、有靠山
- **负面象**：依赖、被动等待、印过重则惰性、母缘事件、文书纠纷

### 偏印（枭神）
- **人物**：继母、偏门师傅、宗教/玄学引路人
- **事物**：偏门技艺、专业冷门、孤独、宗教、灵感、第六感
- **正面象**：专业技术、冷门绝活、悟性高、靠独特技能立身
- **负面象**：孤独、想得多落地难、枭神夺食（才华受阻）、神经敏感、失眠

## 二、宫位象意

四柱各管一段人生 + 一类人事。**宫位被流年/大运冲、合、刑、害、伏吟时，对应的人事就"动"。**

| 宫位 | 人物 | 人生阶段 | 事物领域 | 动起来时应在 |
|---|---|---|---|---|
| **年柱** | 祖辈、父母、长辈、领导 | 0-16 岁、早年 | 原生家庭、根基、出身、社会大背景 | 家族事件、根基变动、早年环境、与长辈/上级关系 |
| **月柱** | 父母、兄弟、同事 | 17-32 岁、青年 | 事业起步、工作平台、社交圈、求学 | 事业变动、跳槽、搬迁、父母/兄弟事件、人际重组 |
| **日支** | 配偶、自己（日干=自身） | 33-48 岁、中年 | **婚姻宫**、自身健康、内在状态 | 婚恋大事、配偶事件、自身健康、中年重大转折 |
| **时柱** | 子女、下属、晚辈 | 49 岁后、晚年 | 子女、事业归宿、晚景、作品/成果 | 子女事件、晚年方向、事业落点、退休、传承 |

**主位 vs 宾位（家内家外，见 methodology Step 7-3）**：日支、时柱 = 家内（自己掌控）；年柱、月柱 = 家外（社会/机会/平台）。

## 三、刑冲合害破的事件画面

脚本算出的地支作用关系，翻译成现实画面：

| 作用 | 能量性质 | 事件画面 | 应在哪类事 |
|---|---|---|---|
| **冲** | 动荡、对撞、移除 | 搬迁、换工作、分离、冲突、远行、手术、关系破裂、旧的去新的来 | 重大变动、位置移动、关系断裂 |
| **合** | 结合、牵绊、拉拢 | 合作、结婚、签约、被拉拢、感情靠近、事情被"合住"拖延 | 关系建立、合作、也可能是被绊住动不了 |
| **半合/拱合** | 局部成势 | 某方面力量被激活、阶段性聚集 | 某五行代表的事被放大 |
| **刑** | 磨削、反复、修理 | 官非诉讼、手术、整顿、反复折腾、被修理、自我消耗（自刑） | 法律/健康/反复性的麻烦、自我纠结 |
| **害（穿）** | 暗损、嫌隙 | 暗中受损、小人、亲近的人之间的嫌隙、出力不讨好 | 隐性损耗、近距离的人际暗伤 |
| **破** | 破坏、解体 | 计划被打破、结构松动 | 变动、不稳 |
| **伏吟** | 同字重叠、内压堆积 | 重复、呻吟、内耗、旧事重演、强烈的内在驱动（P 级来源） | 状态型应期，吉凶看喜忌——可能是主动突破，也可能是反复纠结 |
| **反吟** | 天克地冲 | 剧烈动荡、彻底翻转 | 重大转折、推倒重来 |

**配偶星 + 夫妻宫双动**：当配偶星（男=财、女=官杀）受冲合 **同时** 日支被冲合 → 婚姻类应期信号最强（这正是脚本 🔥 S 级的核心结构）。

## 四、应期如何定时（盲派引动逻辑）

事件"什么时候发生"靠**引动**——原局/大运里某个潜在结构，被流年的字触发：

1. **冲出**：藏干被流年冲出 → 那个十神代表的事浮现（如财库被冲 → 财动）
2. **合来**：流年合住某宫某星 → 那个人事被"拉到眼前"（如流年合日支 → 感情活跃）
3. **透干引动**：流年天干透出某十神 → 该十神当年"显化"（如流年正官透 → 工作/婚姻年）
4. **岁运并临**：流年干支 = 大运干支 → 强应期，旧结构重演
5. **填实**：拱、半合等待"填实"的结构，被流年补全 → 成局应事

**时辰对应期的影响**：时柱（子女宫、晚年）的应期完全依赖时支与流年的作用——所以**确认的子女类、晚年类事件是反推时辰最有力的依据**（见 `--verify-events`）。而日支（婚姻）是固定的，不随时辰变，对定时辰无区分力。

## 五、信号 ↔ 面向 反查表（反馈回路 / 时辰反推用）

命主反馈"某年发生过某类事" → 用此表反查该年**应该**出现什么信号。对得上 → 该解读分支/该时辰成立；对不上 → 怀疑时辰或解读分支（见 methodology 元规则三）。

`bazi_chart.py --verify-events` 用的就是这张表的打分逻辑。

| 命主反馈的面向 | 该年应出现的信号（任一命中即算应期对上） | 时辰区分力 |
|---|---|---|
| **婚恋 / 婚姻 / 感情** | 流年支冲/合/刑/害/伏吟**日支**；配偶星（男=财/女=官杀）被流年天干合或流年透配偶星 | 低（日支固定） |
| **事业 / 工作变动** | 流年透官杀；流年支冲/合/刑**月支**；流年透伤官（事业动荡）；岁运并临于月柱 | 中 |
| **财 / 进财 / 破财** | 流年透财星（进财）；流年透比劫且原局有财（破财）；流年冲/合财星所在支或财库 | 中 |
| **学业 / 升学 / 考证** | 流年透正/偏印；流年合印星支或月支；流年透食伤（发挥/才艺类） | 中 |
| **健康 / 安全** | 流年冲**日支**（身宫）；三刑涉及日支/年支；流年天干剋日主；岁运并临日柱 | 低 |
| **搬迁 / 远行 / 换城市** | 流年冲年支/月支（尤其寅申巳亥驿马）；流年冲日支 | 中 |
| **子女 / 下一代** | 流年冲/合/刑/害/伏吟**时支**；流年透子女星（男=官杀/女=食伤） | **高（时支随时辰变）** |
| **家庭 / 父母 / 长辈** | 流年冲/合/刑年支或月支；流年透印（母）/偏财（父） | 中 |

**关键**：定时辰时，**子女类和晚年类事件权重最高**（时支随时辰变化）；婚姻、健康类事件对定时辰几乎无帮助（日支固定）。所以请命主反馈时优先收集**子女出生/重大事件年份、晚年/事业归宿类事件**。
