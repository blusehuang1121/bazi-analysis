# 内部推理流程（七步，不对外展示）

在写任何对外判断之前，先在内部走完这七步。这是命理分析的"地基"——不走这个流程直接靠直觉出判断，会出现"看到一个特征就跳到结论"的错位。

走完流程后，**只把结论用到对外报告中**，不展示推理过程。

---

## Step 1：调候

先看出生月份和日干，判断命局的寒暖燥湿状态：

- 春生（寅卯月）：木旺，看是否需要火暖、金修
- 夏生（巳午月）：火旺，看是否需要水润、金生
- 秋生（申酉月）：金旺，看是否需要火制、土生
- 冬生（亥子月）：水旺，看是否需要火暖、土制
- 四季月（辰戌丑未）：土旺，看是否需要木疏、金泄

判断："这个命局最缺的'调候之气'是什么？这个气在原局有没有？藏在哪里？"

---

## Step 2：旺衰

判断日主强弱：

- **得令**：日干在月支是否得力（比劫、印星之地为得令）
- **得地**：日干在其他三支是否有根
- **得势**：天干是否有比劫、印星帮扶

综合判断：身旺 / 身弱 / 中和 / 从强 / 从弱 / 化气

---

## Step 3：格局

### 标准取格法（先走一遍）

- 月支本气透出天干 → 取该十神为格
- 月支本气未透但中余气透出 → 取透出者为格
- 都未透出 → 取月支本气为格

### ⚠️ 特殊月令——建禄格 / 羊刃格 / 月刃格

**最容易出错的地方**——月支与日干同党、同气透干时，容易误判成建禄格。

**建禄格**：月支 = 日干的**禄/临官地**（固定位置，不是泛指同党）

| 日干 | 禄地 |
|---|---|
| 甲 | 寅 |
| 乙 | 卯 |
| 丙 / 戊 | 巳 |
| 丁 / 己 | 午 |
| 庚 | 申 |
| 辛 | 酉 |
| 壬 | 亥 |
| 癸 | 子 |

**羊刃格 / 月刃格**：月支 = 日干的**帝旺地**

### 常见取格陷阱

**陷阱 1**：把"月令同党透干"误判成建禄格。例：癸日生亥月，月干透癸——这**不是建禄格**（癸的禄在子，不在亥）。

**陷阱 2**：忽视中气透干。本气未透但中气透了，**优先取中气格**。

**陷阱 3**：两气都透时偏取本气。两个都透时，**本气优先**。

### 特殊结构标注

主格定下来之后，再看是否有这些结构作为补充：
- 伤官见官（格局破坏，但若官藏不透则破坏度有限）
- 杀印相生
- 伤官生财
- 食神制杀
- 从格、化气格（需严格判断，不轻易认从）

**身极旺/身极弱时**，主格要配合"用法"一起讲，例如"伤官格 + 身旺用伤官泄秀""七杀格 + 身弱用印化杀"。

---

## Step 4：用神

基于前三步，决定命局的用神：

- **调候用神**：解决寒暖燥湿问题（出自《穷通宝典》核心思路）
- **扶抑用神**：身弱扶之（印比），身旺抑之（食伤财官）
- **通关用神**：两气交战时，取通关之物
- **病药用神**：命局有"病"（明显的克破），取"药"（解除克破之物）

多个用神时，标注主用神和辅用神。**用神在原局是否得力**（不被冲克、不被合化）是判断命局层次的关键。

## Step 5：大运流年作用关系（必须读脚本输出的表，不要手算）

**重要：脚本已经计算好了"流年作用关系完整表"和"各大运与原局的作用关系"表**——涉及流年/大运作用关系的判断，**必须从这两张表读取，不要自己手算**。

LLM 在干支作用关系上容易出错（漏掉卯戌合、把子卯刑说成合等），而脚本是查表得出的结果，准确可靠。

### 读表方法

**"各大运与原局的作用关系"表**：告诉你每个大运十年的整体背景——哪些位置被冲、合、剋。

**"流年作用关系完整表"**：每一行给出该年的流年作用 + 关键信号。**信号档位是"强度提示"，不是吉凶定性，也不是危机预警**——强度越高代表那年触动越大、变化越大，性质（正/负/混合）由命主当时大运和经营状态决定：
- 🔥 S 级：外触型强能量节点（配偶星动 + 夫妻宫动 + 岁运触发同时成立）
- 🔴 A 级：单项强触动（配偶星或夫妻宫被冲克）
- 🟠 B 级：有动象，未形成完整结构
- 🟡 C 级：辅助参考
- 🔵 D 级：背景

⚠️ **档位是未经数据标定的启发式阈值**（权重和触发门槛是作者直觉编码，不是统计标定）——所以档位本身不是客观精度，**靠元规则三的命主反馈来校准**：过往高档位年命中率高 → 此盘适用；命中率低 → 优先疑时辰。读表时把档位当"值得优先看的强度提示"，不要当"客观危险分级"。

**关键信号列**是这张表最大的价值——已经识别出了"比劫剋财""伤官见官""七杀被合""冲日支""日支伏吟""三刑动妻位"等关键判断点。

### 基于这两张表回答

- 历经大运是助用神还是损用神（结合大运 vs 原局表）
- 当前大运对原局的作用关系（直接读表）
- 当前流年与命局/大运的核心作用点（直接读流年表当年那行）
- 关键转换节点：流年表里**带 ★ 的年份是大运起始年**，前后 1-2 年是过渡期
- 关键年份识别：流年表里**含 🔴 或多个 🟠 标记的年份**是需要重点关注的

### 判断"具体哪几年成为关键年"

脚本的判定逻辑（你读表时也用这个标准）：
- **强作用（关键年）**：流年与日柱/夫妻宫/相关十神宫位形成合、冲、刑、半合、三合
- **次强作用（值得留意）**：流年的天干或地支引动了原局藏干里的关键十神
- **弱作用**：流年与原局没有显著作用，不必特别标注

### 重点关注的流年类型

| 流年作用 | 对应面向 |
|---|---|
| 流年地支冲日支 | 婚姻、家庭、自身健康 |
| 流年合日支 | 关系活跃、感情期 |
| 流年地支冲时支 | 子女、晚年事业 |
| 流年与大运天克地冲 | 重大变动期 |
| 流年引动原局所缺五行/被冲合的关键位置 | 那个面向的对应时间 |

## Step 6：宫位与十神具体应事

最后才看具体面向：

- **年柱**：祖辈、原生家庭、早年环境
- **月柱**：父母、兄弟、青年期、事业起步
- **日柱**：自己（日干=日主）、配偶（日支=婚姻宫）、中年
- **时柱**：子女、晚年、事业归宿

每个面向看：相关十神的状态（透/藏/受冲/受合）→ 推断那个面向的具体特征。

**事件预测的弹药库**：把"宫位 + 十神 + 刑冲合害"翻译成具体现实画面（应在谁、应在什么事、什么时候），查 `blind_imagery.md`（盲派象意参考表）。预测定位下，象意是把命局结构变成可验证判断的关键——没有象意，预测只能停在"今年财运一般"这种无信息量的话。首盘对过去下可验证判断、流年应事、时辰反推的信号↔面向反查，都依赖这份表。

## Step 7：盲派做功效率分析（针对财运、事业层次面向必走）

子平派的"旺衰用神"判断的是命局的静态平衡，但**财运/事业的具体层次**需要用盲派的"做功效率"方法补充——盲派关注的是干支之间能不能形成有效的能量制约，而不是是否平衡。

涉及财运分析时（第三节的财运面向、第八节今年运势的财运部分）、以及涉及事业层次判断时，必须额外走完以下盲派推理。

### 7-1：天干五合做功（盲派做功效率最高的方式）

**这是盲派判断"是否当官 / 是否发财"的第一道筛子**。比地支冲合刑穿、十神生克都更直接、更稳。

**核心规则：**
- **阳干合财**：日主是阳干（甲丙戊庚壬），合的是财星
- **阴干合官**：日主是阴干（乙丁己辛癸），合的是官星
- 合财合官是层次高的做功；克财克官是层次低的做功

**五合 + 化神：**

| 合 | 化神 | 化出 |
|---|---|---|
| 甲己合 | 土 | 化土 |
| 乙庚合 | 金 | 化金 |
| 丙辛合 | 水 | 化水 |
| 丁壬合 | 木 | 化木 |
| 戊癸合 | 火 | 化火 |

**合化的三种结果（看双方力量对比 + 地支化神）：**

| 力量对比 | 地支有化神 | 结果 |
|---|---|---|
| 我弱合方强 | 有 | **合化**（最高效，真正合到对方）|
| 双方相当 | 有 | **合绊**（合而不化，合住但没真合到）|
| 我强合方弱 | 有 | **反化**（合而成自己的五行）|

**判断是否当官（阴日主合官）—— 4 个条件必须全部满足：**
1. 日主合官成功
2. 官是真官（有根有气，不是假神）
3. 官有力（坐实、得令或有源）
4. 官完整（没有受冲、克、穿、合化破坏）

四条全部满足 → 命局结构是"当官的命"

**判断是否发财（阳日主合财）—— 4 个条件必须全部满足：**
1. 日主合财成功
2. 财完整（没有受冲、克、穿）
3. 无搅局星阻挡合化
4. 自身没受制（受制就懒得做事）

**搅局星概念**：第三方剋住合星，让合化破坏。例如甲己合财，但贴邻有庚金克甲 → 甲被庚剋无力合，**庚就是搅局星**。搅局星的应事：命主有合财/合官的意愿和机会，但环境或某个人阻止了实现。

**⚠️ 对外报告时的措辞：**
- ✅ "命局结构是合官的，事业方向上倾向于走承担责任的路径"
- ✅ "财气的结成方式偏直接，做事能转化为收益"
- ❌ "你是当官的命"（这种定性判语禁用）
- ❌ "你必定发财"（绝对化语言禁用）

### 7-2：识别"真神"（命局层次的定盘星）

- 月令藏干和时支藏干是真神的来源（月令约占 50% 能量，时支约占 45-48%）
- 真神不一定是用神——它是"决定命局富贵层级"的关键字
- 检查：真神是否完好？有没有被流年、大运、原局其他位置冲坏或穿坏？

### 7-3：判断"家内 vs 家外"的财官归属

- **家内（主位）**：日支、时柱——属于自己掌控的资源
- **家外（宾位）**：年柱、月柱——代表社会资源、机会、平台
- 做功的本质是把家外的财官**拉回家内**（通过合、冲、穿）
- 财官在家内 = 自己能直接掌握；财官在家外 = 需要"做功"才能拿到

### 7-4：十神暗引判定（修正"缺失维度优先"的原则）

**这是一个关键的隐藏机制**——命局明字看起来缺某个十神，但实际可能被另一个旺神"暗引"出来，命主在那个面向**不是真缺**。

**核心规则：某个十神在月支或时支旺、不受刑穿、不被合化 → 它会"暗引"出下游对应的十神。**

| 旺的十神（在月支/时支）| 暗引出 | 命主实际拥有 |
|---|---|---|
| 食伤旺 | 比劫 | 不缺贵人朋友，关键时候有人帮 |
| 印枭旺 | 官杀 | 暗有权（"制印得权"的根源）|
| 比劫旺 | 印枭 | 有学历或技能（劫财旺→学历；比肩旺→技能）|
| 官杀旺（且原局无财）| 财（"过河财"）| 不缺财（过河财比明财级别更大）|
| 财星旺 | 食伤 | 不缺才华/表达通道 |

**必要条件（必须同时满足）：**
1. 旺的十神在**月支或时支**（最旺点）
2. **不被刑、不被穿**
3. **不被合化掉**（合掉就拉走了，引不出）
4. 该十神是**真神**（有根有气、坐实）

**对"缺失维度优先"原则的修正：**

之前我们 skill 强调"明字缺的五行/十神 = 缺失维度，优先分析"。**加这一层**：
- 看到"明字缺 X 十神" → 不要立刻判定为缺
- **先扫一遍：是否有其他旺神能暗引出 X？**
- 如果暗引在 → 命主在那个面向**不是真缺**，只是表达方式不同
- 如果暗引不在 → 此人在那个面向**确实缺**，缺失维度的判断成立

**例子：**
- 命中比劫不见 + 食伤旺（坐月时、不刑穿）→ 此人贵人朋友不少，不要写"孤独""命犯小人"
- 命中财星不见 + 官杀旺有力 → "过河财"机制，财更大；不要写"求财艰难"
- 命中印不见 + 比劫旺（不刑穿）→ 此人有学历/技能；不要写"学历不高"

**学历方面的细分**：劫财旺→偏向正印（学历）；比肩旺→偏向偏印（技能）。

**⚠️ 对外报告时的措辞**：用"不缺 X""有 X 类型的资源"等中性描述，不说"命主必有学历""注定有贵人"等绝对化语言。

### 7-5：检查"财过河"结构（原局无明显财星时）

- 当原局没有透干的财星，看官杀是否能"当财用"
- 条件：官星没有真正的官性（被穿坏、失去权威感、虚浮无援）
- 这种结构下，**官星反而要被克制才能转化为财富**
- 这是解释"为什么命局看似无财但实际很有钱"的工具

**注**：财过河本质上也是 7-4 十神暗引（官杀暗引财）的一种特殊情况，但因为它在实战中极常见、级别又大，单独列一节强调。

### 7-6：评估"做功效率"

判断流年/大运/原局中的关键作用属于哪一类：

- **高效做功**：申巳、巳酉、卯戌、亥午、卯申（火克金、金木合克这类彻底的能量转换）
- **中效做功**：子丑、寅丑（克中带合，制不彻底）
- **低效做功**：寅亥、辰酉、午戌、卯未（生中带克但效率低）
- **不做功**：申子、午未、亥卯、丑酉、寅午（生而不克，无制约即无功）

做功效率直接对应财富的"结成质量"——但**不要做"层次定盘"式的二元判断**（不说"巨富""贫命"这种话）。只描述"这个结构里财富形成的方式和效率"。

### 7-7：审视"三宝"配置（印、食、禄）+ 比劫真假

**三宝主配置：**

- **印**：权力的行使标志，主"别人为你做事的能力"
- **食**：财的源头，主"才华变现的通道"
- **禄**：身体与俸禄的融合，禄不被穿坏主"基础保障稳"
- 重点看：印是否能护身？食是否被枭印夺？禄是否被刑冲穿破？

**比劫真假应事（贵人 vs 小人的判定）：**

比劫看起来都是"同党、朋友"，但盲派分**真假虚实**，应事完全不同：

| 比劫状态 | 应事 |
|---|---|
| **比劫真神 / 坐实**（透干且本柱地支有根有气）| 真心朋友、关键时候得朋友之助、贵人相助 |
| **比劫假神 / 虚透**（透干但无根无气）| 命犯小人、说假话不办事、用着人朝前用不到人朝后、过河拆桥 |
| **见财官时的比劫**（不管真假）| 倾向于小人（比劫见财 = 夺财；比劫见官 = 抢功）|

**应用到"贵人方向"判断**：
- 命中比劫真神 → 第四节"贵人方向"可以明确写"有真朋友型贵人"
- 命中比劫虚透 → 第四节"消耗模式"应该警惕"合作中的财务边界 / 注意虚假承诺"

### 7-8：警惕"引动比劫夺财"信号（破财应期）

- 流年透出比劫天干、或合动原局比劫
- **最显著的破财应期**——但描述时只说"这一年需要在财务上特别审慎""容易出现合作中的财务调整"，不说"会破财多少"

### ⚠️ Step 7 的使用边界

盲派理论里有一些**威胁性判语**，**全部不允许使用**：

- "克兄弟""克夫""克妻"——禁用
- "行、吃、急、予、瘫"五种残疾判语——禁用
- "贫命""乞丐"等贫富二元定级——禁用，用"财气结成方式比较间接""需要通过持续做事来积累"等中性描述
- "巨富""大富"等绝对量级判定——禁用，用"财气结成的效率较高""命局结构对财富形成是支持的"等相对描述
- 任何把命局结构直接对应到具体灾难/残疾/重大事件的判语——禁用

盲派的**方法论**有价值（五合做功、暗引、做功效率、家内家外、真神理论、三宝、引动比劫等），但**威胁性判语**是命理学里需要被剥离的部分——它们不增加判断的准确性，只增加命主的焦虑。

| Step 6（宫位十神应事）+ Step 7-4 暗引修正"缺失维度" | 三、生活面向分析（除财运外的其他面向）|
| **Step 7 全部**（五合做功、真神、家内外、暗引、财过河、做功效率、三宝+比劫真假、引动比劫） | **三、生活面向分析 → 财运面向（重点）** + **事业层次判断** |
| Step 5（大运流年作用） | 四、当前大运 / 五、下一大运 |
| Step 5 当前流年部分 + Step 7 流年财运部分 | 七、今年的运势分析 |
| Step 7-7 比劫真假 | 四、当前大运 → 贵人方向 / 消耗模式 |
| 全部七步 | 六、整体画像与人生主线 |

## 不要做的事

- 不要把七步推理过程直接列出来给用户看（除非用户明确要求"展示推理过程"）
- **不要在每个判断后都标注"根据《XX》典籍"** —— LLM 容易出现典籍引用错误（编造书名、张冠李戴），所以宁可不引也不要乱引
- 不要因为走了完整推理就让输出变长——推理是为了让结论更准，不是让报告更厚

## 自检

每写一个对外判断时，问自己："这个判断对应的是七步推理里的哪一步结论？"如果说不出来，这个判断是直觉式的，需要回到流程重新核对。

**财运相关的判断额外自检 Step 7 是否走过**：是否考虑了真神状态、家内外归属、做功效率、三宝配置？如果只用了传统派的"财星有力无力"判断，财运分析的颗粒度就不够。

# 元规则

下面两条元规则**贯穿全部七步推理**，是 LLM 在做命理判断时最容易出现的盲点。**每次分析前都要扫一遍这两条**。

---

## 元规则一：隐性结构优先于显性结构（涉及"状态型"判断时）

LLM 在做命理判断时有一个固定偏差——**容易被地支层面的显性作用（冲、刑、合、伏吟）抓住注意力，漏掉天干层面的隐性结构**。这种偏差在判断"哪一年婚姻最不稳""哪一年健康最差""哪一年重大变动最大"这类问题时特别危险——因为这些"状态型"问题往往不是某个显性事件引发的，而是天干层面的能量结构悄悄改变了。

### 区分两类结构

**显性结构（容易看到，权重容易被高估）**：
- 地支冲（子午、丑未、寅申、卯酉、辰戌、巳亥）
- 地支刑（寅巳申、丑戌未、子卯、自刑）
- 地支合（六合、三合、半合、三会）
- 地支伏吟（流年地支与日支相同）
- 大运/流年的十神性质（七杀运、伤官年等）

**隐性结构（容易漏看，但权重往往更高）**：
- 天干合化、合剋、合而不化（比如甲己合让甲木丧失原本的功能）
- 天干透出的十神剋制原局关键十神（比如比劫透干剋制妻星）
- 用神被合住、被剋住（用神功能失效）
- 七杀被合（命主自我约束力的失效）
- 食伤透出剋制官杀（女命婚姻警讯）
- 藏干被冲出（虽然在地支但本质是天干能量的外现）

### 判断原则

- **事件型判断**（哪一年要搬家、哪一年要换工作、哪一年要做手术）→ 显性结构是主导
- **状态型判断**（哪一年婚姻最不稳、哪一年身体最差、哪一年心理最容易失控）→ **隐性结构优先**

### 自检（强制执行）

每次做完地支冲合分析后，**必须再扫一遍天干层面**：

1. 流年天干透出了什么十神？这个十神剋制了原局的哪些十神？
2. 流年天干与原局/大运天干是否发生了合？合后是否让某个关键十神"失效"了？
3. 命主的关键星（男命：妻星 = 财星；女命：夫星 = 官星）今年是被生扶还是被剋制？是被合还是被冲？
4. 命主自身的"约束力"（七杀对身的剋制）今年是加强还是减弱？

如果地支看起来"没什么事"，但天干层面回答上述问题时出现 2 条以上的负面状态——**这一年仍然是关键年，不要因为地支平静就漏掉**。

**自检失败的典型表现**：给出某个面向的"关键年份"时，列出的全是地支冲合的年份，没有任何"天干结构型"的年份。这说明分析停留在了显性结构层，需要重新扫描天干。

## 元规则二：识别"主动型人格 vs 被动型人格"（影响所有面向的判断）

判断任何面向的"动"时，需要先识别命主的人格驱动力类型——**同样的命理信号，主动型和被动型命主的表现形态完全不同**。

这条规则和"隐性结构优先"规则平行使用：那条解决"看哪些信号"；这条解决"怎么解读信号"。

### 主动型人格的核心判据（任意 2-3 条同时出现）

- 官杀有力且透干（尤其七杀透干有制）——内驱力的核心来源
- 财星透干有根——主动求取意愿强
- 食伤生财成局且气势饱满——表达欲和行动力外显
- 比劫不旺，或被官杀有效制约——自我有边界，不易消散
- 日主有根有力——能承接自己的选择
- （参考因素：阳干日主偏向主动，但不作为核心判据）

### 被动型人格的核心判据（任意 2-3 条同时出现）

- 官杀虚弱、被合、藏而不透——缺少外部约束就缺少推动力
- 印星过重（尤其偏印重）——依赖被给予，等待资源就位
- 比劫极旺无官杀有效制约——能量分散难以聚焦
- 食伤旺但无财引导——才华外溢但无方向感
- 财官藏在墓库或被合住——目标感模糊
- （参考因素：阴干日主偏向被动，但不作为核心判据）

### 重要提醒

- **不引入性别预设**——女命的官杀有力依然是主动型，不要因为传统认为女命该顺从就把她归入被动型
- **不带价值判断**——主动型有掌控感的优势但容易固执；被动型有适应性的优势但容易迷失方向
- **判断不到 2 条**：很多命局是混合型，不必强行归类，直接说"这是个偏主动/偏被动的混合型"即可

### 两种类型的"动"模式

**主动型**：
- 决策多由自己发起，内驱力强
- 事业信号触发事业动、关系信号触发关系动，**边界清晰**
- 流年作用直接对应人生事件，判断时按宫位分门别类即可

**被动型**：
- 决策多由外部关系、责任、机会推动
- **事业宫和关系宫的动可以跨界表现**：
  - 月柱（事业宫）被动可能表现为亲密关系变动
  - 日支（配偶宫）被动可能表现为换工作换城市
  - 时柱（子女宫）被动可能表现为事业方向调整

### 跨宫位联动的触发条件（避免过度联动）

被动型命主的关键年份判断里，默认仍按宫位分别看。但出现以下情况时，**主动去对宫位寻找触发源**：

1. 某个宫位看起来"动得不合理"——比如事业宫平静但命主真实经历是大变动
2. 流年透出的天干十神跨越了多个宫位的影响范围
3. 命主自己反馈某个变化"不知道为什么发生"——这种"突然性"往往是跨宫位触发的标志

不要为联动而联动——只在上述情况出现时启用。

### 画像写作的落实

人生画像必须体现"主动型 vs 被动型"的人格驱动力——搞错驱动力类型，整段画像都会偏。

- **主动型命主**的画像调性：聚焦"你的选择""你的判断""你想做的事"——突出内驱力和方向感
- **被动型命主**的画像调性：适度体现"环境塑形 + 你的回应方式"的双向感——但**不要使用"被推着""被命运""被摆布"等宿命感词汇**，而是用"这些机会找上你""这种关系塑造了你的轨迹""你在回应外部信号的过程中长出来的"等中性表达
- **被动型命主**的"事业方向、换城市、重大转变"等节点，可以适度提示"这种变化常常和你身边的关系/责任有关"

**自检方法**：写整体画像前问自己——"这个人的人生轨迹，主要是被他自己的内在选择驱动的，还是在回应周围的关系、机会、责任？"如果答案是后者，**画像里要明显体现这一点**，但不要写成"被命运摆布"的宿命感。

## 元规则三：反馈收敛机制（预测工具的核心引擎）

这条规则决定了这个工具是"越用越准"还是"退化成复读机"。**预测的准确性不靠第一次算得多神，靠的是这条反馈回路。**

### 框架：先验 → 似然 → 后验

- **先验**：第一次出盘，手里只有盲派通则，做纯演绎（首盘）。
- **似然**：命主反馈过往真实经历——这是唯一的"真值信号"。
- **后验**：用反馈收敛盘的解读，得到对这个人更准的判断（复盘）。

每个命主维护一份**命主档案**（格式见 `output_structure.md`「命主档案」节，存于 skill 目录下 `命主档案/`）。首盘无档案 → 纯盲派；反馈进来 → 更新档案；复盘先加载档案 → 在收敛后的解读上预测。

### ⚠️ 铁律：反馈只能调三样，绝不能改盲派通则

单个命主一辈子能确认的大事件就十几个。**用这么点数据去重调盲派的判定规则（改阈值、改做功效率表、自创新规则），是拿噪声当信号，必然过拟合**——结果是命主说什么、工具下次就顺着说什么，看着越来越准，其实只是在复述命主喂进来的东西。这是预测工具最大的自欺陷阱。

反馈**能且只能**调以下三样（都不动通则，只动"这一个盘怎么读"）：

1. **校准命盘本身——尤其定时辰（最高杠杆）。** 盲派对时柱极敏感，时柱错一格整盘应期全错；而出生时间恰恰最不可靠。这是反馈价值最高的一环，见下面「时辰反推」。
2. **在盲派规则允许的多个解读里，收敛到对的那个。** 同一个盘常有 2-3 个都说得通的读法（从财还是身弱用印？这个比劫是真朋友还是小人？主动型还是被动型？）。反馈像似然把先验劈开——错的分支塌掉，对的立起来。
3. **学这个人的"信号 → 面向"表达图谱。** 同一个 S 级结构（比劫剋财+夫妻宫动），在不同人身上应在不同面向（婚恋/合伙/搬迁/健康…）。反馈告诉你这个命主的信号往哪个面向走、他是"逢信号必出事"的高敏型还是"只有强信号才动"的钝感型。（信号↔面向对照见 `blind_imagery.md` 文末反查表。）

### 未命中时的处理顺序（强制）

某个高信号年（🔥/🔴 或 S/P 级）命主反馈"没发生什么" → **按此顺序排查，绝不跳到"改通则"：**

1. **先怀疑时辰**——时柱错会连锁错应期，是最常见原因。启动时辰反推。
2. **再怀疑解读分支**——是不是格局/用神/主动被动型选错了支，换一支重读。
3. **最后才考虑命主特殊原因**（如那年确实平淡、或事件发生在内心未外显）。
4. **永远不要因为对不上就修改盲派通则。**

若过往 ≥2 个强信号年都对不上：**整盘的 S/P 与应期判断降权，并对命主明示**——"本盘的强能量节点识别在你身上吻合度不高，下面的判断请打折看，建议核对出生时辰"。不要在一个可能错的盘上继续盖预测。

### 时辰反推（定时辰的具体方法）

**触发场景**：出生时间不确定 / 首盘列了多个候选时柱 / 过往应期对不上。

**用脚本（推荐）**：
```bash
python scripts/bazi_chart.py -d "1990-03-15 12:00" -g 男 --verify-events "2003:学业,2009:婚恋,2012:子女,2018:事业"
```
- 脚本枚举 12 个候选时柱（日柱固定，仅时辰待定），按确认事件的"应期对齐分"排名。
- 可用 `--hour-candidates "寅,卯"` 缩小候选范围（已知大致时段时）。
- 输出最佳候选 + 逐事件命中明细 + 区分力提示。

**关键认知——哪些事件能定时辰**：
- **时支随时辰变，所以子女类、晚年/事业归宿类事件区分力最高**——请优先向命主收集子女出生年份、晚年重大事件。
- **婚姻、健康类事件对定时辰几乎无用**——它们落在日支（夫妻宫/身宫），而**日支不随时辰改变**，对所有候选时柱同分。它们能验证整盘大方向，但不能区分时辰。
- 脚本会在证据缺乏时支敏感事件时主动提示"区分力弱，建议补充子女类事件"。

**分数接近的多个候选**：都保留为候选时辰，结合命主对盘的体感（性格、长相、六亲状况）综合判断，不要只看分数。时辰定下后写入命主档案。

### 首盘的隐藏任务：把反馈面铺到最大

既然反馈是引擎，首盘就不能只顾预测，还要**故意制造可被证伪的钩子**：

- **对过去要敢下具体、可验证的判断**（过去可查）——"某段大运换运前后经历过一次事业方向大转""早年家中某位长辈对你影响很重"。命主一句"对/不对"就是训练信号。**含糊的首盘喂不出任何修正**，这是很多命理分析永远不会变准的根因。
- **未来仍只给条件窗口**（S/P 节奏 + 节气月），不报点。
- **时辰不确定时，首盘并列列出候选时柱及各自读法**，等过往事件来定。

**一句话**：首盘越敢在"过去"这一侧具体，整个系统收敛得越快。
