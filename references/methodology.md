# 内部推理流程（七步，不对外展示）

在写任何对外判断之前，先在内部走完这七步。这是命理分析的"地基"——不走这个流程直接靠直觉出判断，会出现"看到一个特征就跳到结论"的错位。

走完流程后，**只把结论用到对外报告中**，不展示推理过程。

---

## Step 1：调候

先看出生月份和日干，判断命局的寒暖燥湿状态：

- 春生（寅卯月）：木旺，看是否需要火暖、金修
- 夏生（巳午月）：火旺，看是否需要水润、金生
- 秋生（申酉月）：金旺，看是否需要火制、土生
- 冬生（亥子月）：水旺，看是否需要火暖、土制
- 四季月（辰戌丑未）：土旺，看是否需要木疏、金泄

判断："这个命局最缺的'调候之气'是什么？这个气在原局有没有？藏在哪里？"

---

## Step 2：旺衰

判断日主强弱：

- **得令**：日干在月支是否得力（比劫、印星之地为得令）
- **得地**：日干在其他三支是否有根
- **得势**：天干是否有比劫、印星帮扶

综合判断：身旺 / 身弱 / 中和 / 从强 / 从弱 / 化气

---

## Step 3：格局

### 标准取格法（先走一遍）

- 月支本气透出天干 → 取该十神为格
- 月支本气未透但中余气透出 → 取透出者为格
- 都未透出 → 取月支本气为格

### ⚠️ 特殊月令——建禄格 / 羊刃格 / 月刃格

**最容易出错的地方**——月支与日干同党、同气透干时，容易误判成建禄格。

**建禄格**：月支 = 日干的**禄/临官地**（固定位置，不是泛指同党）

| 日干 | 禄地 |
|---|---|
| 甲 | 寅 |
| 乙 | 卯 |
| 丙 / 戊 | 巳 |
| 丁 / 己 | 午 |
| 庚 | 申 |
| 辛 | 酉 |
| 壬 | 亥 |
| 癸 | 子 |

**羊刃格 / 月刃格**：月支 = 日干的**帝旺地**

### 常见取格陷阱

**陷阱 1**：把"月令同党透干"误判成建禄格。例：癸日生亥月，月干透癸——这**不是建禄格**（癸的禄在子，不在亥）。

**陷阱 2**：忽视中气透干。本气未透但中气透了，**优先取中气格**。

**陷阱 3**：两气都透时偏取本气。两个都透时，**本气优先**。

### 特殊结构标注

主格定下来之后，再看是否有这些结构作为补充：
- 伤官见官（格局破坏，但若官藏不透则破坏度有限）
- 杀印相生
- 伤官生财
- 食神制杀
- 从格、化气格（需严格判断，不轻易认从）

**身极旺/身极弱时**，主格要配合"用法"一起讲，例如"伤官格 + 身旺用伤官泄秀""七杀格 + 身弱用印化杀"。

---

## Step 4：用神

基于前三步，决定命局的用神：

- **调候用神**：解决寒暖燥湿问题（出自《穷通宝典》核心思路）
- **扶抑用神**：身弱扶之（印比），身旺抑之（食伤财官）
- **通关用神**：两气交战时，取通关之物
- **病药用神**：命局有"病"（明显的克破），取"药"（解除克破之物）

多个用神时，标注主用神和辅用神。**用神在原局是否得力**（不被冲克、不被合化）是判断命局层次的关键。

---

## Step 5：大运流年作用关系（必须读脚本输出的表，不要手算）

**重要：脚本已经计算好了"流年作用关系完整表"和"各大运与原局的作用关系"表**——涉及流年/大运作用关系的判断，**必须从这两张表读取，不要自己手算**。

LLM 在干支作用关系上容易出错（漏掉卯戌合、把子卯刑说成合等），而脚本是查表得出的结果，准确可靠。

### 读表方法

**"各大运与原局的作用关系"表**：告诉你每个大运十年的整体背景——哪些位置被冲、合、剋。

**"流年作用关系完整表"**：每一行给出该年的流年作用 + 关键信号：
- 🔥 S 级 极高危：配偶星损 + 夫妻宫动 + 岁运同时触发
- 🔴 A 级 高危：配偶星或夫妻宫被严重冲克
- 🟠 B 级 警示：有动象但未形成完整闭环
- 🟡 C 级 辅助：可辅助解释，不单独定吉凶
- 🔵 D 级 背景：只作参考

**关键信号列**是这张表最大的价值——已经识别出了"比劫剋财""伤官见官""七杀被合""冲日支""日支伏吟""三刑动妻位"等关键判断点。

### 基于这两张表回答

- 历经大运是助用神还是损用神（结合大运 vs 原局表）
- 当前大运对原局的作用关系（直接读表）
- 当前流年与命局/大运的核心作用点（直接读流年表当年那行）
- 关键转换节点：流年表里**带 ★ 的年份是大运起始年**，前后 1-2 年是过渡期
- 关键年份识别：流年表里**含 🔴 或多个 🟠 标记的年份**是需要重点关注的

### 判断"具体哪几年成为关键年"

脚本的判定逻辑（你读表时也用这个标准）：
- **强作用（关键年）**：流年与日柱/夫妻宫/相关十神宫位形成合、冲、刑、半合、三合
- **次强作用（值得留意）**：流年的天干或地支引动了原局藏干里的关键十神
- **弱作用**：流年与原局没有显著作用，不必特别标注

### 重点关注的流年类型

| 流年作用 | 对应面向 |
|---|---|
| 流年地支冲日支 | 婚姻、家庭、自身健康 |
| 流年合日支 | 关系活跃、感情期 |
| 流年地支冲时支 | 子女、晚年事业 |
| 流年与大运天克地冲 | 重大变动期 |
| 流年引动原局所缺五行/被冲合的关键位置 | 那个面向的对应时间 |

---

## Step 6：宫位与十神具体应事

最后才看具体面向：

- **年柱**：祖辈、原生家庭、早年环境
- **月柱**：父母、兄弟、青年期、事业起步
- **日柱**：自己（日干=日主）、配偶（日支=婚姻宫）、中年
- **时柱**：子女、晚年、事业归宿

每个面向看：相关十神的状态（透/藏/受冲/受合）→ 推断那个面向的具体特征。

---

## Step 7：盲派做功效率分析（针对财运面向必走）

子平派的"旺衰用神"判断的是命局的静态平衡，但**财运的具体层次**需要用盲派的"做功效率"方法补充——盲派关注的是干支之间能不能形成有效的能量制约，而不是是否平衡。

涉及财运分析时（第三节的财运面向、第七节今年运势的财运部分），必须额外走完以下盲派推理。

### 7-1：识别"真神"（命局层次的定盘星）

- 月令藏干和时支藏干是真神的来源（月令约占 50% 能量，时支约占 45-48%）
- 真神不一定是用神——它是"决定命局富贵层级"的关键字
- 检查：真神是否完好？有没有被流年、大运、原局其他位置冲坏或穿坏？

### 7-2：判断"家内 vs 家外"的财官归属

- **家内（主位）**：日支、时柱——属于自己掌控的资源
- **家外（宾位）**：年柱、月柱——代表社会资源、机会、平台
- 做功的本质是把家外的财官**拉回家内**（通过合、冲、穿）
- 财官在家内 = 自己能直接掌握；财官在家外 = 需要"做功"才能拿到

### 7-3：检查"财过河"结构（原局无明显财星时）

- 当原局没有透干的财星，看官杀是否能"当财用"
- 条件：官星没有真正的官性（被穿坏、失去权威感、虚浮无援）
- 这种结构下，**官星反而要被克制才能转化为财富**
- 这是解释"为什么命局看似无财但实际很有钱"的工具

### 7-4：评估"做功效率"

判断流年/大运/原局中的关键作用属于哪一类：

- **高效做功**：申巳、巳酉、卯戌、亥午、卯申（火克金、金木合克这类彻底的能量转换）
- **中效做功**：子丑、寅丑（克中带合，制不彻底）
- **低效做功**：寅亥、辰酉、午戌、卯未（生中带克但效率低）
- **不做功**：申子、午未、亥卯、丑酉、寅午（生而不克，无制约即无功）

做功效率直接对应财富的"结成质量"——但**不要做"层次定盘"式的二元判断**（不说"巨富""贫命"这种话）。只描述"这个结构里财富形成的方式和效率"。

### 7-5：审视"三宝"配置（印、食、禄）

- **印**：权力的行使标志，主"别人为你做事的能力"
- **食**：财的源头，主"才华变现的通道"
- **禄**：身体与俸禄的融合，禄不被穿坏主"基础保障稳"
- 重点看：印是否能护身？食是否被枭印夺？禄是否被刑冲穿破？

### 7-6：警惕"引动比劫夺财"信号（破财应期）

- 流年透出比劫天干、或合动原局比劫
- **最显著的破财应期**——但描述时只说"这一年需要在财务上特别审慎""容易出现合作中的财务调整"，不说"会破财多少"

### ⚠️ Step 7 的使用边界

盲派理论里有一些**威胁性判语**，**全部不允许使用**：

- "克兄弟""克夫""克妻"——禁用
- "行、吃、急、予、瘫"五种残疾判语——禁用
- "贫命""乞丐"等贫富二元定级——禁用，用"财气结成方式比较间接""需要通过持续做事来积累"等中性描述
- "巨富""大富"等绝对量级判定——禁用，用"财气结成的效率较高""命局结构对财富形成是支持的"等相对描述
- 任何把命局结构直接对应到具体灾难/残疾/重大事件的判语——禁用

盲派的**方法论**有价值（做功效率、家内家外、真神理论、引动比劫等），但**威胁性判语**是命理学里需要被剥离的部分——它们不增加判断的准确性，只增加命主的焦虑。

## 走完七步后，结论如何映射到报告

| 七步推理 | 对应的报告章节 |
|---|---|
| Step 1-4（调候/旺衰/格局/用神）+ Step 7 真神判断 | 二、整体命局观察 → 命局核心特征 |
| Step 2-3（日主/十神特性） | 二、整体命局观察 → 人格画像 |
| Step 6（宫位十神应事） | 三、生活面向分析（除财运外的其他面向） |
| **Step 7 全部**（做功效率、家内外、三宝、引动比劫） | **三、生活面向分析 → 财运面向（重点）** |
| Step 5（大运流年作用） | 四、当前大运 / 五、下一大运 |
| Step 5 当前流年部分 + Step 7 流年财运部分 | 七、今年的运势分析 |
| 全部七步 | 六、整体画像与人生主线 |

## 不要做的事

- 不要把七步推理过程直接列出来给用户看（除非用户明确要求"展示推理过程"）
- **不要在每个判断后都标注"根据《XX》典籍"** —— LLM 容易出现典籍引用错误（编造书名、张冠李戴），所以宁可不引也不要乱引
- 不要因为走了完整推理就让输出变长——推理是为了让结论更准，不是让报告更厚

## 自检

每写一个对外判断时，问自己："这个判断对应的是七步推理里的哪一步结论？"如果说不出来，这个判断是直觉式的，需要回到流程重新核对。

**财运相关的判断额外自检 Step 7 是否走过**：是否考虑了真神状态、家内外归属、做功效率、三宝配置？如果只用了传统派的"财星有力无力"判断，财运分析的颗粒度就不够。

# 元规则

下面两条元规则**贯穿全部七步推理**，是 LLM 在做命理判断时最容易出现的盲点。**每次分析前都要扫一遍这两条**。

---

## 元规则一：隐性结构优先于显性结构（涉及"状态型"判断时）

LLM 在做命理判断时有一个固定偏差——**容易被地支层面的显性作用（冲、刑、合、伏吟）抓住注意力，漏掉天干层面的隐性结构**。这种偏差在判断"哪一年婚姻最不稳""哪一年健康最差""哪一年重大变动最大"这类问题时特别危险——因为这些"状态型"问题往往不是某个显性事件引发的，而是天干层面的能量结构悄悄改变了。

### 区分两类结构

**显性结构（容易看到，权重容易被高估）**：
- 地支冲（子午、丑未、寅申、卯酉、辰戌、巳亥）
- 地支刑（寅巳申、丑戌未、子卯、自刑）
- 地支合（六合、三合、半合、三会）
- 地支伏吟（流年地支与日支相同）
- 大运/流年的十神性质（七杀运、伤官年等）

**隐性结构（容易漏看，但权重往往更高）**：
- 天干合化、合剋、合而不化（比如甲己合让甲木丧失原本的功能）
- 天干透出的十神剋制原局关键十神（比如比劫透干剋制妻星）
- 用神被合住、被剋住（用神功能失效）
- 七杀被合（命主自我约束力的失效）
- 食伤透出剋制官杀（女命婚姻警讯）
- 藏干被冲出（虽然在地支但本质是天干能量的外现）

### 判断原则

- **事件型判断**（哪一年要搬家、哪一年要换工作、哪一年要做手术）→ 显性结构是主导
- **状态型判断**（哪一年婚姻最不稳、哪一年身体最差、哪一年心理最容易失控）→ **隐性结构优先**

### 自检（强制执行）

每次做完地支冲合分析后，**必须再扫一遍天干层面**：

1. 流年天干透出了什么十神？这个十神剋制了原局的哪些十神？
2. 流年天干与原局/大运天干是否发生了合？合后是否让某个关键十神"失效"了？
3. 命主的关键星（男命：妻星 = 财星；女命：夫星 = 官星）今年是被生扶还是被剋制？是被合还是被冲？
4. 命主自身的"约束力"（七杀对身的剋制）今年是加强还是减弱？

如果地支看起来"没什么事"，但天干层面回答上述问题时出现 2 条以上的负面状态——**这一年仍然是关键年，不要因为地支平静就漏掉**。

**自检失败的典型表现**：给出某个面向的"关键年份"时，列出的全是地支冲合的年份，没有任何"天干结构型"的年份。这说明分析停留在了显性结构层，需要重新扫描天干。

---

## 元规则二：识别"主动型人格 vs 被动型人格"（影响所有面向的判断）

判断任何面向的"动"时，需要先识别命主的人格驱动力类型——**同样的命理信号，主动型和被动型命主的表现形态完全不同**。

这条规则和"隐性结构优先"规则平行使用：那条解决"看哪些信号"；这条解决"怎么解读信号"。

### 主动型人格的核心判据（任意 2-3 条同时出现）

- 官杀有力且透干（尤其七杀透干有制）——内驱力的核心来源
- 财星透干有根——主动求取意愿强
- 食伤生财成局且气势饱满——表达欲和行动力外显
- 比劫不旺，或被官杀有效制约——自我有边界，不易消散
- 日主有根有力——能承接自己的选择
- （参考因素：阳干日主偏向主动，但不作为核心判据）

### 被动型人格的核心判据（任意 2-3 条同时出现）

- 官杀虚弱、被合、藏而不透——缺少外部约束就缺少推动力
- 印星过重（尤其偏印重）——依赖被给予，等待资源就位
- 比劫极旺无官杀有效制约——能量分散难以聚焦
- 食伤旺但无财引导——才华外溢但无方向感
- 财官藏在墓库或被合住——目标感模糊
- （参考因素：阴干日主偏向被动，但不作为核心判据）

### 重要提醒

- **不引入性别预设**——女命的官杀有力依然是主动型，不要因为传统认为女命该顺从就把她归入被动型
- **不带价值判断**——主动型有掌控感的优势但容易固执；被动型有适应性的优势但容易迷失方向
- **判断不到 2 条**：很多命局是混合型，不必强行归类，直接说"这是个偏主动/偏被动的混合型"即可

### 两种类型的"动"模式

**主动型**：
- 决策多由自己发起，内驱力强
- 事业信号触发事业动、关系信号触发关系动，**边界清晰**
- 流年作用直接对应人生事件，判断时按宫位分门别类即可

**被动型**：
- 决策多由外部关系、责任、机会推动
- **事业宫和关系宫的动可以跨界表现**：
  - 月柱（事业宫）被动可能表现为亲密关系变动
  - 日支（配偶宫）被动可能表现为换工作换城市
  - 时柱（子女宫）被动可能表现为事业方向调整

### 跨宫位联动的触发条件（避免过度联动）

被动型命主的关键年份判断里，默认仍按宫位分别看。但出现以下情况时，**主动去对宫位寻找触发源**：

1. 某个宫位看起来"动得不合理"——比如事业宫平静但命主真实经历是大变动
2. 流年透出的天干十神跨越了多个宫位的影响范围
3. 命主自己反馈某个变化"不知道为什么发生"——这种"突然性"往往是跨宫位触发的标志

不要为联动而联动——只在上述情况出现时启用。

### 画像写作的落实

人生画像必须体现"主动型 vs 被动型"的人格驱动力——搞错驱动力类型，整段画像都会偏。

- **主动型命主**的画像调性：聚焦"你的选择""你的判断""你想做的事"——突出内驱力和方向感
- **被动型命主**的画像调性：适度体现"环境塑形 + 你的回应方式"的双向感——但**不要使用"被推着""被命运""被摆布"等宿命感词汇**，而是用"这些机会找上你""这种关系塑造了你的轨迹""你在回应外部信号的过程中长出来的"等中性表达
- **被动型命主**的"事业方向、换城市、重大转变"等节点，可以适度提示"这种变化常常和你身边的关系/责任有关"

**自检方法**：写整体画像前问自己——"这个人的人生轨迹，主要是被他自己的内在选择驱动的，还是在回应周围的关系、机会、责任？"如果答案是后者，**画像里要明显体现这一点**，但不要写成"被命运摆布"的宿命感。
