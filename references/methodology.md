# 内部推理流程（六步，不对外展示）

在写任何对外判断之前，先在内部走完这六步。这是命理分析的"地基"——不走这个流程直接靠直觉出判断，会出现"看到一个特征就跳到结论"的错位。

走完流程后，**只把结论用到对外报告中**，不展示推理过程。

---

## Step 1：调候

先看出生月份和日干，判断命局的寒暖燥湿状态：

- 春生（寅卯月）：木旺，看是否需要火暖、金修
- 夏生（巳午月）：火旺，看是否需要水润、金生
- 秋生（申酉月）：金旺，看是否需要火制、土生
- 冬生（亥子月）：水旺，看是否需要火暖、土制
- 四季月（辰戌丑未）：土旺，看是否需要木疏、金泄

判断："这个命局最缺的'调候之气'是什么？这个气在原局有没有？藏在哪里？"

---

## Step 2：旺衰

判断日主强弱：

- **得令**：日干在月支是否得力（比劫、印星之地为得令）
- **得地**：日干在其他三支是否有根
- **得势**：天干是否有比劫、印星帮扶

综合判断结论类型：
- 身旺 / 身弱 / 中和
- 从强格 / 从弱格 / 化气格（需严格判断，不要轻易认从）

---

## Step 3：格局

### 标准取格法（先走一遍）

- 月支本气透出天干 → 取该十神为格（正官格、偏财格等）
- 月支本气未透但中余气透出 → 取透出者为格
- 都未透出 → 取月支本气为格

### ⚠️ 特殊月令——建禄格 / 羊刃格 / 月刃格

这三个格不是按"本气透干"来取的，而是按**月支与日干的特殊关系**来取。**最容易出错的地方就是这里**——月支与日干同党、同气透干时，很容易误判成建禄格。

**建禄格**：月支 = 日干的**禄/临官地**（注意："禄"是固定的位置，不是泛指同党）：

| 日干 | 禄地（月支为此即建禄格）|
|---|---|
| 甲 | 寅 |
| 乙 | 卯 |
| 丙 / 戊 | 巳 |
| 丁 / 己 | 午 |
| 庚 | 申 |
| 辛 | 酉 |
| 壬 | 亥 |
| 癸 | 子 |

**羊刃格 / 月刃格**：月支 = 日干的**帝旺地**（阳干叫羊刃，阴干争议较大，部分流派承认月刃）：

| 日干 | 帝旺地（月支为此即羊刃/月刃）|
|---|---|
| 甲 | 卯 |
| 乙 | 寅（阴干逆行，争议）|
| 丙 / 戊 | 午 |
| 丁 / 己 | 巳（阴干逆行，争议）|
| 庚 | 酉 |
| 辛 | 申（阴干逆行，争议）|
| 壬 | 子 |
| 癸 | 亥（阴干逆行，争议）|

### 常见取格陷阱（自检）

**陷阱 1：把"月令同党透干"误判成建禄格**

例：癸日生亥月，月干透癸（比肩）。这**不是建禄格**——癸的禄在子，不在亥。亥对癸是帝旺，争议下可视为月刃，**正格仍按"亥中藏甲透时干"取伤官格**。

陷阱：看到"月令水透干癸"就直觉叫"建禄"，错。先查日干禄地表。

**陷阱 2：忽视中气透干**

月令本气没透但中气透了，**优先取中气格**（中气有透就比本气没透"显"）。例：亥月本气壬不透但中气甲透时干 → 取**伤官格**。

**陷阱 3：本气、中气都透时偏取本气**

两个都透时，**本气优先**（本气力量大于中气）。

### 特殊结构单独标注（在主格之外）

主格定下来之后，再看是否有这些结构作为补充：
- 伤官见官（格局破坏，但若伤官见官的官藏不透则破坏度有限）
- 杀印相生
- 伤官生财
- 食神制杀
- 从格、化气格（需严格判断，不轻易认从）

**身极旺/身极弱时，主格要配合"用法"一起讲**——例如"伤官格 + 身旺用伤官泄秀""七杀格 + 身弱用印化杀"。光说格名不说用法，分析就停留在表面。

---

## Step 4：用神

基于前三步，决定命局的用神：

- **调候用神**：解决寒暖燥湿问题（出自《穷通宝典》核心思路）
- **扶抑用神**：身弱扶之（印比），身旺抑之（食伤财官）
- **通关用神**：两气交战时，取通关之物
- **病药用神**：命局有"病"（明显的克破），取"药"（解除克破之物）

多个用神时，标注主用神和辅用神。**用神在原局是否得力**（不被冲克、不被合化）是判断命局层次的关键。

---

## Step 5：大运流年作用关系

基于前四步，看：

- 各大运是助用神还是损用神
- 当前大运对原局造成什么作用关系（合、冲、刑、害）
- 当前流年与命局、大运的核心作用点
- 关键转换节点：大运交脱前后 1-2 年的过渡期（必须标注）

**判断"具体哪几年成为关键年"的方法：**

- **强作用关系（关键年）**：流年与日柱/夫妻宫/相关十神宫位形成合、冲、刑、半合、三合
- **次强作用关系（值得留意）**：流年的天干或地支引动了原局藏干里的关键十神
- **弱作用（不必特别标注）**：流年与原局没有显著作用

**重点关注的流年类型：**

| 流年作用 | 对应面向 |
|---|---|
| 流年地支冲日支 | 婚姻、家庭、自身健康 |
| 流年合日支 | 关系活跃、感情期 |
| 流年地支冲时支 | 子女、晚年事业 |
| 流年与大运天克地冲 | 重大变动期 |
| 流年引动原局所缺五行/被冲合的关键位置 | 那个面向的对应时间 |

---

## Step 6：宫位与十神具体应事

最后才看具体面向：

- **年柱**：祖辈、原生家庭、早年环境
- **月柱**：父母、兄弟、青年期、事业起步
- **日柱**：自己（日干=日主）、配偶（日支=婚姻宫）、中年
- **时柱**：子女、晚年、事业归宿

每个面向看：相关十神的状态（透/藏/受冲/受合）→ 推断那个面向的具体特征。

---

## 走完六步后，结论如何映射到报告

| 六步推理 | 对应的报告章节 |
|---|---|
| Step 1-4（调候/旺衰/格局/用神） | 二、整体命局观察 → 命局核心特征（术语层）|
| Step 2-3（日主特性/十神特性） | 二、整体命局观察 → 人格画像 |
| Step 6（宫位十神应事） | 三、生活面向分析 |
| Step 5（大运流年作用） | 四、当前大运 / 五、下一大运 / 七、重要年份 / 八、今年运势 |
| 全部六步 | 六、整体画像与人生主线 |

---

## 不要做的事

- 不要把六步推理过程直接列出来给用户看（除非用户明确要求"展示推理过程"）。
- **不要在每个判断后都标注"根据《XX》典籍"** —— LLM 容易出现典籍引用错误（编造书名、张冠李戴），所以宁可不引也不要乱引。
- 不要因为走了完整推理就让输出变长——推理是为了让结论更准，不是让报告更厚。

## 自检

- 每写一个对外判断时，问自己："这个判断对应的是六步推理里的哪一步结论？"
- 如果说不出来，这个判断是直觉式的，需要回到流程重新核对。
